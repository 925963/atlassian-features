## Description

Organizations using Focus and Talent can now view Talent funds and labor rate data directly in the Positions tab of Focus. This feature requires labor rates to be configured in Talent, and provides greater visibility into funds data for improved workforce planning. Explore more about connecting Atlassian apps to Focus. To get started with this change Configure labor rates in the Talent app if not already set up.
	Enable the feature for Focus in Focus admin settings under App connections for Tale
