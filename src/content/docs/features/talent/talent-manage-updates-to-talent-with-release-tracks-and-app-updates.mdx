## Description

Talent can now be put on the bundled release track, so you can have more time to prepare and plan for changes to the Talent app. What are release tracks? To schedule monthly changes for Talent: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Apps > Release management > Release tracks.
	Look for Talent in the list of supported apps and select the change track option.
	Choose the bundled release track to schedule monthly updates.
