## Description

Managers and admins can now save time by updating focus area allocations for multiple positions at once. This update provides greater flexibility for workforce planning, allowing you to select several positions and assign them to a focus area in a single action, rather than updating each one individually. To bulk update multiple positions at once: Select Manage positions from the left navigation.
