## Description

The saved views feature is now available in Talent, allowing users to save a specific table setup — including filters, sorting, columns, and groupings — as a reusable view with its own URL. This helps you personalize and streamline repetitive tasks as well as allows you to quickly return to that exact configuration later and share it with others so they see the same thing you do. To get started with this change: Open the Talent app and go to a supported page (All positions, Sub-positions).
