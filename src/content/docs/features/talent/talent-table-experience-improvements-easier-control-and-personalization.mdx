## Description

This change improves how you work with table columns in Talent by making it faster to show, hide, reorder, and resize columns across most views, so you can keep your tables focused on the details that matter most to your team. To get started with this change: Open a table view in Talent.Select the new Columns button to search, pin, reorder, or show/hide the fields you need.
