## Description

Talent is now integrated into Atlassian global search, making it easier to find Talent-related information across your Atlassian apps. This enhancement helps you quickly locate Talent content from across the enterprise platform. It also enables the global search capability within Talent. Open any Atlassian Cloud app and access the global search from your navigation bar.
