## Description

You can now upload an optional external position key field to Talent as a standard field and use it to refer to positions in your external human resources information system (HRIS) data source, such as Workday. You can search and sort by this field, and re-order or hide this column in Talent. The Position ID field has also been renamed to Position key to make it more readable.
