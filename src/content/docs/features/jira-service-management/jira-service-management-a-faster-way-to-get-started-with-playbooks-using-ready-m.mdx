## Description

You can now jumpstart your playbooks experience with a library of out-of-the-box templates. These templates are designed to help teams quickly discover use-cases, import playbooks, and customize processes covering common scenarios across IT Operations, IT Services, Human Resources, and more – no need to start from scratch. To use the new playbook templates: Go to Project settings.
	Select Playbooks.
	Select Templates. Choose a template to preview its details and steps.
