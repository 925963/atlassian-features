## Description

Jira Service Management now supports Date, Date/Time, Time, Number, Text, User, and Users field types in advanced conditional logic for forms. This allows you to create more dynamic forms by showing or hiding fields based on specific dates or numerical values provided by users. Open the form you want to edit in Jira Service Management.
