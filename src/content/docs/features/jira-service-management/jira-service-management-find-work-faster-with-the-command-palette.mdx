## Description

We've added new navigation items to the command palette to help you quickly access your work items. You can now view your open work items and work items reported by you more easily. To access the new navigation items: Open the command palette:
	
		Mac: Command + K
		Windows: Ctrl + K
	
	
	Search for the command View my open work items or View work items reported by me.
	Select the command to navigate to view those work items.
