## Description

The Delay until component now supports a Webhook received event. This lets you pause automation rules until an external system triggers a webhook and chosen conditions are met, helping you better coordinate work across tools. To use this event in your automation rules: Navigate to Automation in your Jira or Confluence space.
	Create or edit an automation rule.
	Add the new Webhook received event to the Delay until component.
	Configure and save your automation rule.
