## Description

Customers on a Standard plan can now use HR integrations in their automation rules. Discover more about integrating HR components such as Workday, Entra, Docusign, Okta, and Workato when building automation rules. To start using these components: Select Space settings then Automation.
	Choose from the available HR components when creating or building your rules.
