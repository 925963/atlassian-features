## Description

You can now trigger an agent session for a work item by clicking the Agents button and selecting an agent. To get started with this change Open a work item in your Jira software or business space.
	Click on the Agents button to see a list of agents available to collaborate in Jira.
	Use the search field to narrow down your results.
	When you select an agent, it will immediately get to work, and you can view the session from the Agents section of the work item.
