## Description

We recently announced a big change: team-managed fields will be visible in the Fields pages in Jira Admin. Alongside that change, team-managed field will be held in the Deleted Fields tab of the Fields page for 60 days before deletion (like all other fields). This way, you can restore them if necessary. This action is only available to those with the Administer Jira permission.
