## Description

A new Journeys chart has been added to the metrics and charts that can be found on the project summary page. Read about the project summary page in Jira Service Management. Navigate to your service project, then select Summary in your project’s sidebar. At the bottom of the page, you'll see the Journeys bar chart showing the count of in-progress journeys by journey name.
