## Description

Jira Product Discovery is an Atlassian app that allows people to create custom roadmaps. You can add or edit information such as dates, assignees, and more. You may now see a recommendation in your project menu about Jira Product Discovery roadmaps. You can select this recommendation, add Jira Product Discovery, and create a new project with a custom roadmap template. Only admins will see this recommendation.
