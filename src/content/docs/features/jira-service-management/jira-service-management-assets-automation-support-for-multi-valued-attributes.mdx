## Description

Jira Service Management has updated the Attribute value changed automation trigger to support multi-valued attributes. This allows you to trigger automation rules when changes occur in attributes that contain more than one value, providing more comprehensive coverage for your asset management workflows. To get started with this change: Go to Project settings, then Automation.
	Select Create rule and choose the Attribute value changed trigger.
	Select the object type and the multi-valued attribut
