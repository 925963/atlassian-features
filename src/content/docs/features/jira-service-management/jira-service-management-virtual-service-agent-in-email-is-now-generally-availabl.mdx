## Description

We’re excited to announce that the virtual service agent in email is now generally available. You can activate the virtual service agent in email so that your customers can find the help they need. When your customers send requests to the email address connected to your project, the virtual service agent responds using Atlassian Intelligence answers. Find out more about using virtual service agent in email
