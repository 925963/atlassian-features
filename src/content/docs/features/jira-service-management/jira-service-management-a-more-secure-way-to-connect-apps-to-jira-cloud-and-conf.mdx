## Description

We are excited to announce OAuth 2.0 as a new authentication type for Cloud Application Links. This authentication type offers an industry-standard authentication protocol that enables secure, modern, and reliable connections between Atlassian products. To start using OAuth 2.0 Application Links in Jira and Confluence: Go to Jira or Confluence.
	Go to the Jira/Confluence settings.
	Find Application links.
	Follow the prompts to create a new Application Link using OAuth 2.0.
