## Description

We are deprecating the “Create feature flag in LaunchDarkly” action in Atlassian Automation on June 30, 2025. As a result, the rules linked to LaunchDarkly and this action will cease to function as intended after this date. To prevent rule failures, all rules employing this action must be reconfigured on or before June 30, 2025.
