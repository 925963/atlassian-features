## Description

We’ve updated the look and feel of onboarding popups and feature spotlights across Jira Service Management to provide a more modern and consistent experience. These changes include a visual refresh of the benefits modal in knowledge base settings, canned response spotlights in the issue view, and suggested topic spotlights in the knowledge base hub. To get started with this change: Go to Knowledge base settings or an issue view to see the updated spotlights.
