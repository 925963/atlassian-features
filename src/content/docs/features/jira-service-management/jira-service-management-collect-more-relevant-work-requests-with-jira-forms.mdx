## Description

Collect only the relevant details from your stakeholders. By creating conditions, you can now show specific questions based on responses to previous questions. To begin creating conditions on a form, you’ll need project admin access: Open a Jira project and navigate to Forms in the project navigation.
	Select an existing form or select + Create form.
	Add a priority field, or any custom multi-select or single-select fields to the form builder.
	Select that field from the form builder to edit.
