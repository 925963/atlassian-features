## Description

You can now filter work items in Jira search using more granular status options. This update gives you finer control when narrowing down your search results by status. To use the new granular status filter in Jira search: Go to Jira and open the search experience for work items.
	Find the Status filter in the search options.
	Select from the expanded list of status options to refine your search results.
