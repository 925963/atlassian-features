## Description

Admins can now recalculate service-level agreements (SLAs) for a single issue directly from the issue view in Jira Service Management. This update provides a quick way to repair SLAs that may have started at the incorrect time or displayed inaccurate values without needing to use a REST API or change global configurations. This feature is currently available for company-managed projects. To get started with this change: Open an issue in a company-managed project.
