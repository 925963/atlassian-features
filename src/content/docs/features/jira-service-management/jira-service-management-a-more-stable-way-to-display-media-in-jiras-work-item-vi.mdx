## Description

Media in Jira’s work item view editor will now use pixel widths instead of relative sizing. This change helps prevent media from resizing unexpectedly while content loads, providing a more consistent experience. Resize using the new handle: Resize your images using the drag handle located on the sides of the images. Your image will proportionally grow or shrink based on how much you drag it in and out.
