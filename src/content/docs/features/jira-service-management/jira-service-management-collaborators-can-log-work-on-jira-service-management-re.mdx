## Description

Collaborators in Jira Service Management can now log work, and edit or delete their own work log entries on requests. This helps teams track time and contributions more accurately, without requiring collaborators to have a Jira Service Management agent license. Open a request in a Jira Service Management project where you're added as a collaborator.
	Select Log work to add your time entry.
	Edit or delete your own work log entries as needed.
