## Description

You can now allow Slack apps, workflows, or webhooks to trigger Jira Service Management ChatOps responses using schedule aliases. Previously, ChatOps only responded to alias messages sent by users. Now, when an app posts a message with an alias, you can allowlist it directly from the message shortcut menu.
