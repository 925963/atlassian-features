## Description

Project admins in Jira Service Management company-managed projects can now set a custom date to start processing emails for IMAP and OAuth mail channels. This self-service option lets you control which emails are converted into requests, reducing the need to contact Atlassian support and helping you manage incoming requests more efficiently. Go to your service project and select Project settings.
