## Description

You can attach media to comments in Atlassian tools like Jira and Confluence. Previously, if you did this the media would show on the comment using a placeholder. Now, when you attach media to a comment it will show an icon and relevant information instead of the placeholder. This means you’ll be able to quickly and easily understand what the attachment is.
