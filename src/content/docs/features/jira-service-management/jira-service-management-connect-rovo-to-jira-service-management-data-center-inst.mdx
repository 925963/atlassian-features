## Description

Rovo Search now supports connecting Jira Service Management Data Center instances to your Atlassian Cloud site. This enables organizations to synchronize and index Service Desk project types, allowing Service Desk work items to appear alongside Jira Software data in Rovo Search for more comprehensive results. To connect Rovo to Jira Service Management Data Center instances: Create an application tunnel or configure an allowlist (for private networks).
