## Description

You can now assign a dedicated Manage Versions permission in Jira and Jira Service Management projects without giving full project admin access. Users with this permission can create, release, archive, merge, edit or delete versions within a specific project only. This applies to company-managed projects in Jira and Jira Service Management, and team-managed projects in Jira only.
