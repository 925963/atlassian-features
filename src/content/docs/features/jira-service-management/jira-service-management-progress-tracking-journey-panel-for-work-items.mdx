## Description

A new progress panel in Jira Service Management helps you track the status of a journey for a work item. When you view a journey parent work item, you can now see exactly how much work has been completed and what is still in progress. To get started with this change: Open a work item that serves as a journey parent.
	Locate the new journey panel within the work item view.
	Review the progress of all steps in that journey.
