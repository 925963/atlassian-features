## Description

You can now set dependencies using individual statuses on a journey, not just status categories. This gives you more control and lets you map out your process in greater detail, specifying exactly when certain work should be created. To set up dependencies in your journey: Select Add  anywhere on the journey type between two work items where you’d like to add a dependency and select Add dependency.
	Select the work item you’d like the dependency to apply to.
