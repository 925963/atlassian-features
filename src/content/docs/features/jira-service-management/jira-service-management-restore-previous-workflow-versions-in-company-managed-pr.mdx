## Description

Earlier versions of a workflow can now be restored directly from the workflow editor in company-managed projects. Previous versions are available for up to a month, allowing you to recover recent configuration and undo changes if needed. To restore a previous workflow version from the workflow editor: Select the More actions menu, then Restore workflow to an earlier version.
	Choose the version you want to restore and select Restore.
