## Description

Jira Service Management has expanded the capabilities of the Attribute value changed automation trigger. You can now configure multiple attributes within a single rule, support both single-valued and multi-valued attributes, and detect changes in inherited attributes for child object types. To get started with this change: Go to Project settings, then Automation.
