## Description

Project admins can now add or remove views directly from the space navigation, giving them more flexibility to customize the space to suit their team's need. To add a view, select the plus icon + in your space navigation and choose which view you want to add. To remove a view, select the tab options menu (...) next to the tab name, and then select Remove.
