## Description

Agents and administrators can now mention any request participant in a work item or comment, even if the participant has not previously commented. This makes it easier to collaborate and keep all relevant people informed. To get started with this change: As an agent or admin, open a Jira Service Management work item.
	In the work item's description or comment, type @ followed by the name of any request participant to mention them.
