## Description

Jira Service Management now provides clearer, more accurate error messages when users or agents try to add an email address that doesn’t have access to a service project. These improved messages help admins and agents quickly identify if an issue is caused by project permissions or configuration settings rather than a generic system error. To get started with this change: Go to your service project and select an existing request.
