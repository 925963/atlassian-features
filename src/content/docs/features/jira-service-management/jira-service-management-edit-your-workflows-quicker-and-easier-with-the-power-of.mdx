## Description

In Jira sites that have Rovo enabled, the New Workflow Editor now includes an AI Workflow builder agent that can help add, change, or delete statuses and transitions in a workflow. It can also answer questions about the key differences between the old and new workflow editor. While the agent is still learning, you can prompt it to add or remove the following rules from an existing workflow: Restrict who can move a work item
