## Description

You can now use Automation smart values in the Basic AQL Builder component. This is an opt-in feature, so your current setup will not change unless you choose to enable it. To start using Automation smart values in the Basic AQL Builder: Go to the Basic AQL Builder component in your Automation setup.
	Look for the option to enable smart value support.
	Opt in to start using smart values in your queries.
