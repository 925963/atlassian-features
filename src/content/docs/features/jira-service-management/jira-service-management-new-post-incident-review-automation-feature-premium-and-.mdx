## Description

You can now use the new post-incident review automation feature in Jira Service Management plans (Premium and Enterprise). This helps your team capture learnings and follow-up actions as part of incident resolution. To use the new post-incident review in automation: Select Space settings, then Automation.
	Create or edit an automation rule.
	Add the post-incident review automation to your rule.
	Edit the title and description using smart values or fields from the incident.
	Publish your rule.
