## Description

The Promotion feature in Bitbucket Pipelines deployments is being deprecated. This change simplifies the deployments experience and improves reliability by reducing complexity and technical debt in Bitbucket Pipelines, paving the way for future enhancements to deployments. (You can still deploy the same way you do today from pipelines, but you will no longer be able to promote an existing deployment between environments from the Deployments dashboard.)
