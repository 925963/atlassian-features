## Description

To make setting up Rovo Dev easier, Bitbucket workspace admins can now enable code reviews for multiple repositories in a workspace at once. To enable Rovo Dev for all eligible repositories in a workspace: In your Bitbucket workspace, select Settings from the main navigation, then Workspace settings.
	Select Rovo Dev from the sidebar.
	Activate Rovo Dev if it’s not already enabled, then select Activate for all eligible repositories.
