## Description

Bitbucket now enforces limits on the number and size of pipeline variables, helping teams maintain efficient and reliable pipelines. This change ensures that pipeline configurations remain manageable and reduces the risk of errors caused by excessive or oversized variables. To get started with this change Go to your Bitbucket repository.
	Select Pipelines from the sidebar.
	Ensure the total number and size of custom pipeline variables for a pipeline do not exceed the enforced limits.
