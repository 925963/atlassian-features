## Description

In the past, variables in Bitbucket Pipelines were scoped to only operate within script sections of a Pipelines workflow. This limited their usefulness and restricted users from being able to fully parameterize their workflows. Today we are releasing an enhancement that will allow variables to be used within any section of a Pipelines .yaml file, and not just within scripts.
