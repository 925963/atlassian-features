## Description

You can now use the new state step condition to control whether a step should run, based on a boolean expression using context variables. Both state and changesets conditions are now evaluated right before each step starts, giving you a consistent and flexible way to manage step execution. To use the new dynamic step condition: Open your Bitbucket repository and go to Pipelines.
