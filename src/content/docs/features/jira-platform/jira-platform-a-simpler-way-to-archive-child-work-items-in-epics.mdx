## Description

This update introduces the ability to archive an epic's child work items when you're archiving the epic. You can now easily manage and tidy up your epics by archiving related work items that are no longer active or needed. To archive child work items in your epics (ensure you have archive permissions): Open an epic containing child work items.
	Select Archive for the epic (either on the board, or within the work item view).
	Select the option to archive child work items.
	Select Archive again.
