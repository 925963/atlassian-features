## Description

Jira Enterprise admins can now grant users access to custom templates for creating spaces. This update streamlines space creation, enabling teams to use custom templates created by admins, reducing manual requests and configuration errors. To get started with this change: Create a new custom template or open an existing one to edit.
	Select Choose who can create new spaces from this template to grant access to users or groups.
	Users with access can select this template when creating a new space
