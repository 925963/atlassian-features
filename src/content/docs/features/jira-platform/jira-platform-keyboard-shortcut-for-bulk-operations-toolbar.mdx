## Description

Jira now includes a keyboard shortcut to quickly interact with the bulk operations toolbar when it is visible on the Backlog, List, and Issue Navigator views. • Mac: Cmd+B 
• Windows: Ctrl+B This shortcut is also listed in the keyboard shortcuts modal, making it easier for users to discover and use bulk operations efficiently via the keyboard.
