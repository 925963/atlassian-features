## Description

We've added syntax highlighting to code blocks in the Jira editor, making it easier to write, read, and share code in your Jira content types. To use code blocks with syntax highlighting in Jira content types: Open or create a Jira work item.
	Go to any text editing section, such as description or comment.
	Select the "Insert" menu in the editor toolbar.
	Choose "Code block" from the options.
	Select your programming language to enable syntax highlighting.
