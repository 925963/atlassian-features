## Description

When this feature rolls out, admins will be able to: associate a new field to all of the field configurations used by a scheme without leaving the page, and
	view and edit which field configurations that field is a part of via the More actions (•••) menu on the Fields page This work is part of our ongoing efforts to improve Fields management in Jira.
