## Description

Teams can now create Confluence whiteboards directly from their Jira backlog and import backlog issues, making it easier to plan and prioritize work. This update helps teams quickly access in-context visual collaboration spaces without leaving their planning workflow. Go to your Jira backlog. If you’re an admin and your site doesn’t have Confluence yet, you’ll see a prompt to add Confluence and start a whiteboard.
	If you have at least three issues and Whiteboards is enabled, you’ll see a prompt
