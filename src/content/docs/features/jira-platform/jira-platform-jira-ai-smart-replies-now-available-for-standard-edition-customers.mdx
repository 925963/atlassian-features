## Description

You can now use AI-powered smart replies in the Standard edition of Jira, as long as AI features are enabled. This update gives you automated, context-aware reply suggestions, so you can respond to work items faster and more efficiently. To start using AI smart replies in Jira: Open any work item in Jira where AI features are enabled.
	Look for the AI smart replies suggestions below the comment box.
	Select a suggested reply to insert it into your response, then edit or send as needed.
