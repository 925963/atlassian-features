## Description

You can now filter your Rovo search results in Jira by label, reporter, and status. This update makes it easier to find the content you need by narrowing down your search using these new filter options. To use the new filters in Rovo search within Jira: Open Jira and go to the Rovo search experience.
	Enter your search query as usual.
	Use the new filter options for label, reporter, and status to refine your results.
