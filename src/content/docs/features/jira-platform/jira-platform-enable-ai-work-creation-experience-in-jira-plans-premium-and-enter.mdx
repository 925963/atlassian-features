## Description

You can now use the same AI-powered work creation experience from the backlog view in Jira Plans. This lets you generate work items from your Confluence content, Loom videos, and images without switching contexts. To use the AI work creation feature in Jira Plans: Open your Jira Plan.
	Create a work item inline, between work items, or as a child work item.
