## Description

This update introduces a new Copy link button in the invitation screen, making it easier to quickly copy and share invitation links with others. To use the new Copy link button: Open the invitation screen where you invite others to your workspace or content.
	Look for the new Copy link button.
	Select Copy link to copy the invitation link to your clipboard.
	Share the copied link with anyone you want to invite.
