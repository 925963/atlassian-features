## Description

Microsoft is deprecating the Azure DevOps OAuth platform and requiring all apps to use Microsoft Entra ID for authorization by 2026. To ensure your Azure DevOps data continues to appear in Jira without interruption, Jira admins must reauthorize the Azure DevOps for Jira app with Microsoft Entra ID. If authorization isn’t completed before the deprecation deadline, your Azure DevOps data will stop syncing with Jira until reauthorization is complete.
