## Description

Forge platform developers and Marketplace Partners can now build apps that run global background scripts in Jira Cloud. This new capability doesn't introduce changes to existing settings or interfaces, but it enables more powerful behind-the-scenes automation, such as sending alerts to all users who need to complete compliance training. To learn more, see Jira global background script.
