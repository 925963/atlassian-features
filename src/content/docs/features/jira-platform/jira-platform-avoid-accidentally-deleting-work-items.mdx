## Description

We’re changing the way users delete work items to help avoid accidental deletions. When you attempt to delete a work item, you must complete a two-step process to confirm the action. To delete a work item: Select the work item you want to delete.
	Select More actions (•••), then select Delete.
	To go ahead with deleting the work item, type “delete” in the text box.
	Select Delete (this option will only be available after you type “delete” in the text box).
