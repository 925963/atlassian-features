## Description

Issues assigned to deactivated or suspended users now display correctly in Jira plans. Previously, these issues appeared as unassigned, which caused confusion and created extra unassigned swimlanes when grouping by assignee. Deactivated users now show with their name and a (Deactivated) label, consistent with how the Jira issue view handles inactive users. To get started with this change: Open a plan in Jira.
