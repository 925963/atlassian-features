## Description

You can now drag and drop field columns in the timeline view to instantly reorder them. You’ll also find a More actions menu in each column header to quickly remove or reorder columns, so you can make adjustments without interrupting your workflow. To reorder columns in your timeline, select and drag a column header to your desired position. For more options, select the More actions (...) menu in the column header to reorder or remove columns.
