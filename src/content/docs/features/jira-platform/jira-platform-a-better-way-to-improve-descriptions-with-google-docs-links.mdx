## Description

You can now use Google Docs links when improving work item descriptions. This update makes it easier to bring in content from your favorite document tool to help you write better work item descriptions. To use Google Docs links in the description improver: Open a work item where you want to improve the description.
	Paste a Google Docs link into the description field.
	Select Improve description.
	Follow the prompts to generate an improved description using your linked Google Doc.
