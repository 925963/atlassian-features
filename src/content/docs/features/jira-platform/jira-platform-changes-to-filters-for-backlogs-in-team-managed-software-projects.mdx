## Description

We've redesigned the filtering experience in backlogs for team-managed software projects. You can now filter work items in the Backlog view. We’ve also added two new filters - Status and Priority - allowing you to quickly refine your backlog view to focus on what’s more important. Read more at Changes to filters in team-managed software boards.
