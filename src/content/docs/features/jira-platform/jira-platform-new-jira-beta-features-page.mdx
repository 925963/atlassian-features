## Description

Jira admins can now manage experimental features from a new Jira Beta Features page in Jira Admin Settings. This update centralizes feature management by moving control from individual user settings to a dedicated admin interface in Jira Cloud, ensuring consistent experiences across your site. To get started with this change: Go to Settings.
	Select System.
	Select Beta Features to view and manage available beta features.
