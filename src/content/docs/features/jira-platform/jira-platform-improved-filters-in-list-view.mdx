## Description

We’ve made it easier to find and focus on the work that matters in Jira with a simplified filter experience. Now, when you’re in the All work tab (soon to be merged into a single List view), you’ll see a streamlined filter option that brings quick filters and advanced options like JQL into a single, modern component.
