## Description

We've added more types when you filter search results in Jira, giving you more flexibility and control when finding what you need. Simply select from Board, Dashboard, Filter, or Plan to start filtering. To use these new filters in search: Select the filter button and choose any Jira app.
	Select from the new type options: Board, Dashboard, Filter, or Plan to refine your search result.
