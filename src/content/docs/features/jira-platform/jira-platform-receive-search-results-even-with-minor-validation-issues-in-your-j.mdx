## Description

When you run a search in Jira’s list view with minor validation issues in your JQL, you’ll receive matching work items along with a clear validation message, helping you fix your query as needed. To view work items using a JQL query: Go to the list view in Jira
	Enter a JQL query to search for work items.
	Review and refine your JQL query as needed.
