## Description

When performing a search using in Jira using the Quick find, you’ll now see suggested search terms before you enter any text. These suggestions, such as “Show me all of my in-progress work items” and “Recent open items assigned to me,” help you quickly find relevant issues and get started faster. Go to Jira Cloud.
	Select Quick find (the search bar at the top of the screen).
