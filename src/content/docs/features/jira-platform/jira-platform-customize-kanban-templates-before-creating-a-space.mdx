## Description

You can now customize kanban templates for team-managed spaces before creating the space. Set up work item types and column names in advance so the new space is tailored to your team's needs. To customize a kanban template for your team: In the template gallery, select the kanban team-managed space template.
	Enter the details for your new space.
	Customize work item types and column names to match how your team works.
