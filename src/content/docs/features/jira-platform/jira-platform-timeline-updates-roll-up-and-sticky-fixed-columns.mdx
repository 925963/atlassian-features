## Description

We’re excited to announce the rollout of the new roll-up and sticky (fixed) column features for the timeline in business spaces. This update is all about making space tracking smoother, more intuitive, and less manual—so you can focus on what matters most. To explore this change: Go to your business space's timeline.
	Create or open a parent work item with child work items.
	Assign start and due dates to child work items to see roll-up values on the parent.
	Use the new assignee and status colum
