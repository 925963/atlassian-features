## Description

Add more functionality to your automation rule, allowing you to link multiple connected processes together in a single rule. This can help reduce the number of rules you need to maintain. We’re beginning with Advanced components such as Branch at the same time, which enables execution of multiple components in parallel. Rule groups enable execution of multiple components in specific segments. Available for Cloud Premium and Enterprise editions only. Find the new Advanced component type available
