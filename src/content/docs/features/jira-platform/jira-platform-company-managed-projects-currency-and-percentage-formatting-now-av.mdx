## Description

We’ve added new formatting options for number fields in company-managed projects. You can now format any number field to show a number, currency symbol, or percentage symbol. This feature enhances the presentation of numerical data and provides an easy way to track information like budgets, estimates, costs, and more. To try out a number field with currency or percentage formatting: From Jira admin settings, select Fields.
