## Description

We've redesigned the setup experience when you create a new plan. This guided flow makes it easier for you to choose work sources, teams, and field columns so your plan is aligned with how your teams work from the start. To create a new plan: Select the plus icon + next to Plans in your navigation.
	Follow the steps and add relevant details to complete your plan set up.
