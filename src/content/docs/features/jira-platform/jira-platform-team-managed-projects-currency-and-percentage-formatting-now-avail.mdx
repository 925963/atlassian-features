## Description

We’ve added new formatting options for number fields in team-managed projects. You can now format any number field to show a number, currency symbol, or percentage symbol. This feature enhances the presentation of numerical data and provides an easy way to track information like budgets, estimates, costs, and more. You must be an admin to edit fields.
