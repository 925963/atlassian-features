## Description

We recently released the ability to view fields from team-managed spaces in Fields pages in Jira Admin. This release has a few quick improvements to that: 1. you can now sort columns according to team-managed fields using the Can be used in column. In this space, you'll see either All projects or Team-managed projects only  2. Jira admins can now remove fields from private projects; previously, this was not possible.
