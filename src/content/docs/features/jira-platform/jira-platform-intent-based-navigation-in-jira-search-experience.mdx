## Description

Jira Cloud now uses intent detection in the search experience to automatically choose and visually highlight the best destination based on your search query. This helps users find relevant information faster and more efficiently. To get started with this change: Go to Jira Cloud.
	Type your query into the search bar.
	Review the highlighted destination in the search header and select it to continue.
