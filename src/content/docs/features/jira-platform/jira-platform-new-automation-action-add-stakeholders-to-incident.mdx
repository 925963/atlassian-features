## Description

You can now automatically add stakeholders to incidents using the new Add stakeholders to incident action. This helps streamline communication and ensures the right people are kept informed during incident response. To give the new automation component a try: Navigate to either project automation (in Jira) or global automation.
	Create a new rule, or edit an existing rule.
	Add the new Add stakeholders to incident action to your automation rule.
	Configure the action to specify which stakeholder
