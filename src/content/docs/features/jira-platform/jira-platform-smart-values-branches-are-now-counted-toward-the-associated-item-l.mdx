## Description

From 31 October 2025 onwards, smart value branches will be considered a part of an automation rule’s associated items. This means that branched items from a smart value will be counted toward the associated items limit, which is currently 5000. Rules that breach this limit will be disabled and won't be able to run. To prevent this, check any rules that contain a large amount of smart value branches and ensure they won't exceed the associated items limit before 31 October 2025. View automation se
