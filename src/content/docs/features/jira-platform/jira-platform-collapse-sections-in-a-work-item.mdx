## Description

You can now collapse sections in a work item to reduce scrolling and highlight key information. To collapse sections in a work item: Open a work item in Jira.
	Find the section you want to collapse, such as Subtasks or Activity.
	Select the arrow next to the section header to expand or collapse the section.
