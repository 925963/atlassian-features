## Description

In Jira, you can now add a comment whenever you flag a work item, no matter where you do it. Previously, you could only add a comment from views like the board, backlog, or the command palette in the work item view. This update adds the same comment prompt when you add or remove a flag from the More actions (•••) menu in the work item view, giving your team consistent context about why work is flagged as at risk. To get started with this change: In Jira, open the work item you want to flag.
