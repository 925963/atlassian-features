## Description

We’ve added new settings that let site admins and space admins control who can request access to spaces. With these site‑ and space‑level settings, you can turn the Request access button on or off where it appears, helping reduce unwanted access requests and making permissions easier to manage. Go to Settings in Jira, then select System in the admin settings.
