## Description

If someone has access to both Jira and Confluence but has not used Confluence in the last 28 days, they will now see a flag in Jira encouraging them to rejoin their team in Confluence. This flag includes a direct link to Confluence Home and an option to dismiss the message. To see and interact with this new flag in Jira: Log in to Jira with an account that also has access to Confluence.
