## Description

We’re making the project deletion process easier to follow. When a project is moved to trash and marked for deletion, you have a 60 day window in which to restore a project. After that, the project will be permanently deleted. With this change, if you don’t require the 60 day grace period, you can select 'Delete now' from the project trash list menu. A ‘Deleting now’ message confirms the project deletion process is underway.
