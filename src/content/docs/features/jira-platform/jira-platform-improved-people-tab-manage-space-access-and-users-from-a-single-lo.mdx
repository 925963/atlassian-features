## Description

Manage space access and user controls more efficiently with a restructured people and access settings view. The user list is the primary focal element, and access settings are behind a settings cog for a cleaner workspace while keeping essential controls within reach. For company-managed spaces: Go to your company-managed Jira space, then select the People tab.
	Select the Settings cog to view and manage space access and request settings.
