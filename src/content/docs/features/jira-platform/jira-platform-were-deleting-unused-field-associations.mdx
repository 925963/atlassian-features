## Description

Starting today (June 16, 2025), we’re beginning the process of removing unused field associations from your field configuration schemes. We expect this process to take several weeks from today. Once finished, this will improve the performance and stability of your Jira instances. If this causes any issues in your projects, we have documented the steps for adding fields back in this Atlassian community announcement.
