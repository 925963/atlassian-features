## Description

Jira Cloud has updated three classic reports to improve date formatting and provide a more consistent experience. The Average Age, Created vs Resolved, and Recently Created Work Items reports now use the latest interface, resolving previous issues with how dates were displayed for different regions. To get started with this change: Go to your company-managed space in Jira.
	Select Reports from the sidebar.
	Choose Average Age, Created vs Resolved, or Recently Created Work Items.
