## Description

You can add sibling work items directly in your timeline using the Create work item inline, making it easier for you to manage related work items in one place. To create an inline sibling work item in your timeline: Find a work item in your timeline where you want to add a sibling work item.
	Hover in-between two work items in your plan and select the Create work item inline that appears.
	Fill in the details of your sibling work item and select Create to save you changes.
