## Description

A Quickstart panel will now automatically appear in plans for new users to highlight key actions that help you get started (Premium only). To view the new Quickstart panel in Jira Premium plans: Open your plan in Jira.
	The Quickstart panel will appear if you're new to plans. If you previously dismissed Quickstart, select the Quickstart button to open it.
	Select Show me within each information section to complete the checklist and take a tour of key features.
