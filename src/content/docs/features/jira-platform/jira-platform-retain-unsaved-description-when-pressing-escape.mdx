## Description

When editing a description, pressing the Escape key will now preserve your unsaved changes as a draft instead of discarding them. This update helps prevent accidental data loss and makes it easier to continue editing where you left off. Open any work item in Jira Cloud and start editing the description field.
	Type your changes, then press the Escape key.
	Re-enter edit mode to see your unsaved changes retained as a draft.
