## Description

We’ve decluttered and simplified the work item view. To create a child work item or add any related work, you can: Open any type of work item (not a subtask).
	Select the plus icon  below your work item’s name.
	Select Create child work item.
	Enter a summary for your child work item, and select Create or press enter.
