## Description

In Jira’s work item view, we’ve moved the Vote and Feedback actions into the More actions (•••) menu. Previously, these actions were available outside of the menu. This change simplifies the work item view and makes it easier to focus on key actions. To see this change, navigate to your Jira project. Select a work item to open the work item view, then select More actions (•••).
