## Description

When you’re viewing work items in Jira’s All work tab, you’ll notice an improved keyboard navigation experience that enhances accessibility for all users. To navigate work items with the keyboard: Use arrow keys to move between cells.
	Edit a cell by pressing return (Mac) or enter (Windows). Exit edit mode by pressing esc (Mac and Windows).
	Use home and end keys to jump to the start or end of a row.
	Use page up and page down to quickly move through your list.
