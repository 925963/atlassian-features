## Description

You can now create a new work type or edit an existing one directly from the dropdown when creating work items inline on the software board view. This experiment aims to make managing work types faster and more convenient while you work. To try out the new inline issue type configuration on your software board view: Go to your software board view in Jira Software.
	Select the dropdown for work type when creating an issue inline.
	Choose to create a new work type or edit an existing one directly
