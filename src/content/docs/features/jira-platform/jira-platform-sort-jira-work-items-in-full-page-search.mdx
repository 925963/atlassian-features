## Description

This change lets you sort work items in Jira in the full page search experience when you use Rovo in Jira Cloud, so you can quickly focus on the most relevant results. To get started with this change: Open Jira Cloud.
	Open the Rovo full page search experience and search for work items.
	Use the Sort options in the results view to reorder the list of work items.
