## Description

We’ve updated the All work view so fields now follow the configuration of the space you’re in. The column selector, visible columns, and CSV exports now hide fields that aren’t part of your current Jira space, reducing noise and keeping you focused on the work that’s actually relevant to you. To view this change: Open a space in Jira.
	Select the All work tab.
	Select Configure columns (three vertical columns icon).
	Only columns that are associated with your space will now be appear in the list
