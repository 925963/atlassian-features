## Description

The work item creation experience on the plan timeline is getting a new design. Instead of using a nested dropdown menu, you will now see a popup with fields. This update is designed to address accessibility concerns and make it easier for everyone to create work items. A new Create  button has also been added to group headers, allowing you to create work items directly within a group. To use the new work item creation popup on the plan timeline: Go to your plan timeline in Jira.
