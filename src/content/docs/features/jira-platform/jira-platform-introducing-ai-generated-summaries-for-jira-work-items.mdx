## Description

Jira now offers an AI-generated summary card for work items. This feature helps users quickly understand complex work items by providing a concise overview - including information about its status, key contributors, and next steps - directly in the work item view. Open any Jira work item.
	Select the new Work Item Summary button at the top of the work item, next to Quick Add.
	View your work item's AI-generated summary.
