## Description

Now, when you start typing in Jira's search bar, you’ll see query suggestions that relate to what you’ve entered. Your recent queries will also appear right after these suggestions, making it easier to pick up where you left off. To try out the new query suggestions and recent queries in Jira's quick search: Open Jira and go to the search bar.
