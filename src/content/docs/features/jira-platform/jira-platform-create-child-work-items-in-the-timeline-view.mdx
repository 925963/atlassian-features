## Description

You can now create child work items directly from the parent work item, allowing you to build out your plan in the timeline view. To create a child work item in the timeline view: In the Timeline view, select the parent work item where you want to create a child work item.
	Select the plus icon  next to the summary of the parent work item.
	Fill in details for your child work item and select Create.
