## Description

Fields will no longer be editable in plans if they’re not configured for a space and work item type. This helps you avoid errors when editing and ensures the right fields are used. This change applies to both team and company-managed spaces. To configure fields for a company-managed space: Navigate to the Jira admin panel to review which fields are available.
	To make a custom field available, add it to the relevant space and work item type.
