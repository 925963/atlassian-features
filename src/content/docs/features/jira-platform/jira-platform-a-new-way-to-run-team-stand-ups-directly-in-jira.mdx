## Description

We are adding the option to facilitate a standup with your team inside Jira. This feature lets you run team standups without leaving Jira, making it easier to keep everyone in sync and focused on work items. The stand-up feature in Jira now offers a more robust experience, including participant persistence, a redesigned interface, new customization settings, and a built-in feedback collector.
