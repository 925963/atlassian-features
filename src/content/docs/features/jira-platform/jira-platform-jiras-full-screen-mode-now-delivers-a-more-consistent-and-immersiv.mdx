## Description

We’re extending and refining Jira’s full screen mode, introducing a more consistent, immersive, and productive experience across all major Jira views. To try the updated full screen experience in Jira: Open any major view in Jira, such as a board, backlog, or work item view.
	Select the full screen icon in the top right corner of the view.
	Explore the updated layout and navigation in full screen mode.
