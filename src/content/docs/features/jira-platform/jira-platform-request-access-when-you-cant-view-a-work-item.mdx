## Description

You can now request access to a work directly. This allows you to quickly ask project admins for permission to view a work item or project, so you’re not left at a dead end. To request access to a work item: Open the work item you don’t have permission to view.
	Select Request access on the “You don’t have access to this work item” screen.
	You’ll see a confirmation that your request has been sent to the project admins.
