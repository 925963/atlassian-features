## Description

We've redesigned how rules are added to transitions, streamlining how you build your workflow. Now, when you select a rule, you'll see four different categories: Restrict transition, Request input, Validate details, and Perform actions. To configure a rule to your workflow: Open the workflow editor and select a transition.
