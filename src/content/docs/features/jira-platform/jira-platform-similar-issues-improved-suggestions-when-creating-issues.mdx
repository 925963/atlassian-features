## Description

Save time and reduce duplicate work with more relevant suggestions in Jira. We’ve updated how similar issues are identified when you create a new issue. Instead of suggesting issues while you type, Jira now provides a more accurate list of similar issues once you finish entering the summary and move to the next field. To get started with this change: Select Create in the navigation bar to open the Create issue dialog.
