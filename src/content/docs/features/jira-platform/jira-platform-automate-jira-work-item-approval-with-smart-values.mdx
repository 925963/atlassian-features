## Description

We've introduced new smart values to refer to the manager of a Jira work item reporter or assignee. For example, you could use these smart values to automatically set the approver of a Jira work item to the assignee’s or reporter’s manager. The new smart values include issue.reporter.manager or issue.assignee.manager. Check the manager you want to refer to is set up through User Provisioning.
