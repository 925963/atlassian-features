## Description

We’re introducing a toolbar in the issue navigator so users can update multiple issues even more seamlessly. To use the new toolbar: Select multiple issues from the list view in the issue navigator. You can either:
	
		Use the issue checkboxes.
		Hold Ctrl (Windows) or Command (Mac) while selecting issues
		hold the Shift key to select a group of adjacent issues
