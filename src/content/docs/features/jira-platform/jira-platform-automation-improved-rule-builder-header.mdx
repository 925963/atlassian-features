## Description

We've uplifted the header in the automation rule builder to improve readability and navigation across Jira. This change makes it easier to manage rules on smaller screens and provides a cleaner workspace by moving secondary views into a consolidated menu. To get started with this change: Go to Automation in your Jira app.
