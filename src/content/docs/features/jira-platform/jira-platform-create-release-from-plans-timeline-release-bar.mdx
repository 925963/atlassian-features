## Description

We've updated how releases are created. Now, when you hover over the release bar, you can create a release either through the indicator (appears as a blue dot) or plus icon , making it easier manage and schedule releases without leaving your timeline view. To create a release using the indicator: Hover over your release bar in your plan timeline and select the indicator.
	Select Create release, fill in any required information, and select Create.
