## Description

The Atlassian Rovo Model Context Protocol (MCP) Server is now GA for Jira, Confluence, and Compass. It lets AI tools and IDEs securely read and write Atlassian Cloud data using the MCP, with enterprise‑grade controls, including domain allowlists, IP allowlist support, and audit logs. To start using Atlassian Rovo MCP Server: Configure MCP settings in Atlassian Administration. This includes Atlassian-supported domains, trusted domains added by admins, and security policies.
