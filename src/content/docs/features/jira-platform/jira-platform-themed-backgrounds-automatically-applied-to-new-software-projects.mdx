## Description

When you create a new software project, a default themed background will now be applied automatically. This update brings a consistent and visually engaging look to every new software project from the start. To change your default background: Go to your space in Jira. Then:
	
		In team-managed spaces: Select ••• (more) next to the space name.
		In company-managed spaces: Go to your board, then select ••• (more) next to the board name.
