## Description

This update simplifies channel management and enhances the visibility and usage of the virtual agent across your Jira Service Management projects. Streamlined configuration: For a more efficient admin experience, manage all channel configurations from a single Jira Service Management channel home. Centralized channel visibility: View all connected channels with the virtual agent in one place.
