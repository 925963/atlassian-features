## Description

We are introducing a single, streamlined experience that brings together child work items (like sub-tasks, stories, or tasks) and linked work items (such as blockers, dependencies, or related work items) into one cohesive view. Previously, these were shown in separate panels, which could make it harder to see relationships and manage work efficiently. To find and use the new unified work item section: Open any work item (such as a story, task, or bug) in Jira.
