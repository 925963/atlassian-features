## Description

Gather context in Jira with AI-powered content suggestions from across your connected apps. When you add a web link to a work item, Jira now suggests relevant Confluence pages, whiteboards, and Loom videos, alongside files from third-party apps like Google Drive, Microsoft OneDrive, and Figma. To get started with this change: Open a work item in Jira.
	Select the plus icon (+), then select Add web link.
	In the Web links section, select View suggestions.
