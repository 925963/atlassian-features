## Description

We’ve made it easier to manage more work at once in Jira with the new bulk operations experience. Now, when you select work in the List view or the All work tab, you can select up to 1,000 work items that match your filters with a single click from the bulk operations toolbar, even if they aren’t all loaded on your screen.
