## Description

Atlassian Rovo now provides real-time skill suggestions as you type in chat. This improvement helps you quickly discover and use relevant Rovo agents and skills, saving you time and effort when writing prompts. To get started with this change: Open Rovo Chat.
	Begin typing your prompt in the input field.
	Select a suggested skill or agent from the list that appears to add it to your chat.
