## Description

The first three prompt cards from Rovo's prompt library now appear in fullscreen Rovo Chat, replacing the previous conversation starters. This update makes it easier for users to discover and use helpful prompts when starting a conversation with Rovo. To get started with this change: Open Rovo fullscreen chat.
	View the first three prompt cards displayed at the top of the chat window.
	Select a prompt card to start a conversation with a suggested prompt.
