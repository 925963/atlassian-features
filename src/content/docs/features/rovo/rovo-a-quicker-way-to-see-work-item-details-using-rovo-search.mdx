## Description

Now, when you search for Jira work items using Rovo search, you can view more details instantly by hovering over the result. This update introduces a hover card that displays key information about each work item, helping you decide if it’s the one you need without leaving your search. To use the new hover card in Rovo search: Open Rovo search in Jira.
	Search for a Jira work item.
	Hover over any work item in the results to see the details card.
