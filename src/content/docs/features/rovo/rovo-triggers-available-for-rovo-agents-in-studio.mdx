## Description

We're introducing triggers to Studio to allow you to automate more actions in your Rovo agents. Only people with access to Automation will be able to create triggers in Studio. To use triggers in your agent, Navigate to Agents and select an existing Rovo agent or create a new one.
	Select Triggers.
	Select Add automation trigger to configure and complete your automation.
