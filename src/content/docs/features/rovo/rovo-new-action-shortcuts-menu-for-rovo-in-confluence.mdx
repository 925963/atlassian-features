## Description

We've added a new menu of action shortcuts which you can access by hovering over the Rovo button in Confluence. This update helps users discover and access Rovo features more easily, making it easier to use it in Confluence. To get started: Navigate to a page in Confluence where the Rovo button is available.
	Hover over the Rovo button.
	Select a shortcut from the menu to execute an action.
