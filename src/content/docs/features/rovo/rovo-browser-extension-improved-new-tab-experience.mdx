## Description

The Rovo browser extension now offers a smoother and more consistent experience when you work with Search and Chat in a new tab. Updates include more responsive interface elements and improved management of keyboard focus (for example, when you open Chat in the sidebar). How to install the Rovo browser extension To get started with this change, open a new tab in your browser with the Rovo extension enabled.
