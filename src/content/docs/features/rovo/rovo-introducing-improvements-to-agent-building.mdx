## Description

We are introducing scenarios for Rovo agents in Atlassian Studio. This change allows you add multiple scenarios to an agent you're building. What are scenarios? Each scenario contains its own instructions, actions, and knowledge sources and they're tied to a trigger. That means you can use scenarios to determine how your agent responds in specific situations. Navigate to Studio via the app switcher.
	Edit an existing Rovo agent, or create a new one.
