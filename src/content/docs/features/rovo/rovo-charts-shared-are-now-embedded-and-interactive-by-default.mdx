## Description

When you share a Rovo-generated data chart in your content, it will now automatically appear as an embedded chart instead of a simple link. This makes it easier for everyone to view and interact with shared charts directly in their content. To see embedded charts: Prompt Rovo with a task to visualize a chart with data.
	Select Copy link from the Rovo-generated chart.
	Paste the link into your content. It will automatically display as an embedded, interactive chart.
