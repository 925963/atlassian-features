## Description

If you’re an organization admin, you can give a verified status to Rovo agents. Adding a verified status helps users to identify that the agent is safe, supported, and accurate within your organization. Agents with a verified status will be surfaced in Rovo Search and Chat, as well as in the Browse agents tab, so users can easily find trusted agents. Any agents you verify will appear with a blue checkmark next to their names, and will be listed underneath Verified agents in the Browse agents tab
