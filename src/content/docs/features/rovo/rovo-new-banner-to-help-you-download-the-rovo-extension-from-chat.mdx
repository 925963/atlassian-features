## Description

Rovo now shows a banner above the chat input, in both the sidebar and full page chat, that let's you to download the Rovo extension if you don’t already have it, so you can start using Rovo from anywhere. To see and use the new banner: Open Rovo chat in either the side panel or full page view.
	Send at least one message.
	Look for the banner above the chat input box.
	Select the call-to-action button on the banner to install the Rovo extension, or dismiss the banner.
