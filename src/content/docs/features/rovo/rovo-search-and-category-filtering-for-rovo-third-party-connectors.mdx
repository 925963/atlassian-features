## Description

We are introducing search and category filtering to help with finding third-party connectors to use with Rovo. This change allows you to search for connectors by name, and has a category filter, which allows you to filter the connector list by the type of connector (such as Collaboration, Communication, or Development.) To access the new experience for Rovo, you must have Rovo enabled within your organization. Go to the Cloud Admin section in your Atlassian app.
