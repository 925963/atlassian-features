## Description

Rovo Skills are now available for all customers using Rovo Chat. Skills help Rovo understand the steps it needs to take to complete a task, and what an example of ‘good’ would look like for that task. Skills also transform a natural language prompt into consistent, reliable action. For example, instead of asking Rovo to “write me something about this release”, you can use the skill /summarize-release that understands releases, work items, and how to talk about those ideas to your stakeholders.
