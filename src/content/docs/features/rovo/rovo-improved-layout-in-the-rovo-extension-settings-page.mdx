## Description

The Settings page in the Rovo extension has been redesigned - making admin-controlled site settings more visible and easier to understand. The updated layout and navigation help users quickly see which features are enabled or disabled for each site. To get started with this change: Open the Rovo extension in your browser.
	Select Settings to view the updated settings page.
