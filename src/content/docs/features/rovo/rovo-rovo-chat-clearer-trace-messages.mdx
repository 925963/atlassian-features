## Description

Rovo Chat trace messages have been redesigned to provide a faster and more transparent experience while Rovo is thinking. You can now see exactly which Atlassian apps and third-party apps are being searched, along with a preview of the sources used to generate your response. To get started with this change: Open Rovo Chat and enter a prompt that requires searching across multiple apps.
	Observe the traces returned to see real-time progress.
	Look for product-specific icons to identify which apps
