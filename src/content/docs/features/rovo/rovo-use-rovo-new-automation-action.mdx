## Description

You can now use a free-form AI prompt in your automation rules across Atlassian apps. This new action allows you to send instructions to Rovo to process data, generate content, or summarize information as part of your automated workflows. The AI-generated response is captured as a smart value, rovoResponse, which you can use in subsequent steps of your rule, such as adding a comment to a work item or sending a notification.
