## Description

When you use Think Deeper, Rovo is now better at following complex conversations and providing multi-step reasoning. With this, Rovo delivers more consistent and thoughtful responses through improved contextual memory. To use Think Deeper in a Rovo chat: Open the Customize button below the prompt composer
	Toggle on Think Deeper and type out your question
	Select Submit to receive and review the answer generated
