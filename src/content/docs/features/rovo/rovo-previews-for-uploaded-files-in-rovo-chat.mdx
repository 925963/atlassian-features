## Description

You can now instantly preview files uploaded in Rovo Chat, including PDFs, text files, and images. This update provides visual thumbnails and click-to-preview functionality, making it easier to identify and review uploaded files across Confluence, Jira, and Platform experiences. Upload a PDF, text file, or image in Rovo Chat.
	Select the file thumbnail (for image and text file) to preview its contents directly in the chat window.
