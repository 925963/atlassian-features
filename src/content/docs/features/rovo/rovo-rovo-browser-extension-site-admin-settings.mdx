## Description

Site admins can now access and manage settings for the Rovo browser extension. This update ensures that people with site admin permissions have the necessary access to configure extension options and view announcements, even if they aren't organization admins. This change helps site admins better support their teams by allowing them to manage Rovo settings directly within the browser extension. To get started with this change: Open the Rovo browser extension.
