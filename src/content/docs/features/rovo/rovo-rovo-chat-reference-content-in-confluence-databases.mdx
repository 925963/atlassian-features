## Description

When you select a field, entry, card, or cell, Rovo Chat now visually indicates what content is being referenced in your prompt, giving you more context and control over your AI-powered work in Confluence. To get started with this change: Navigate to a database in Confluence.
	Open the Rovo Chat sidebar or modal.
	Select the content you want to reference, such as a field, entry, card, or cell.
	Enter your prompt in Rovo Chat and see the referenced content visually indicated in the chat interface
