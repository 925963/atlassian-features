## Description

You can now use Rovo to create multiple Jira work items at once from action items and tables on a Confluence page. This update helps you quickly capture and organize tasks, making it easier to manage your work in Confluence. Open a Confluence page in view mode.
	Highlight the action items or table items you want to create in bulk.
	Choose the option to create multiple work items.
	Insert the created work items on the page. Note: this feature has also been rolled out to edit mode for action items
