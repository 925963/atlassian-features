## Description

Opsgenie data residency gives you control over where your Opsgenie account data is hosted. We support data residency moves (moving data across locations) for all Standard, Premium, and Enterprise plans for Opsgenie. To request a data residency move in Opsgenie, go to Settings > App settings > Data residency. Learn more about data residency in Opsgenie
