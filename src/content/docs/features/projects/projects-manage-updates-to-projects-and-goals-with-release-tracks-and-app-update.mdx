## Description

Projects and Goals can now be put on the bundled release track, so you can have more time to prepare and plan for changes to the Projects and Goals apps. What are release tracks? To schedule monthly changes for Projects and Goals: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Apps > Release management > Release tracks.
	Look for Projects or Goals in the list of supported apps and select the change track option.
