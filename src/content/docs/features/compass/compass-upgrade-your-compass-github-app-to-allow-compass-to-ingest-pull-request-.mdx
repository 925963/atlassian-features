## Description

Accept the updated permissions for your Compass GitHub app to authorize Compass to enrich the PR data/insights we show you in Compass. Upgrade your Compass GitHub app: In GitHub, go to the organization settings for the organization that you've connected the Compass app to.
	Select GitHub apps on the left sidebar.
	Select the Review request link.
	Enter your GitHub password.
	Select Accept new permissions.
