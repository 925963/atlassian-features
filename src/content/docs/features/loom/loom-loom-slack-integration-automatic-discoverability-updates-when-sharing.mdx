## Description

The Loom Slack integration now automatically updates the discoverability of any Loom video when it is shared in a direct message, group direct message, or private or public Slack channel. This ensures that recipients can easily discover shared Loom videos within Slack. To get started with this change: Connect your Loom account to Slack using the Loom Slack integration.
