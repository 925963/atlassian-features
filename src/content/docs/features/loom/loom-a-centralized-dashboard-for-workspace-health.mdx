## Description

Loom workspace admins on Premium and Enterprise plans now have access to a centralized Overview dashboard. This new landing page provides immediate visibility into key workspace health metrics, such as ROI data, license allocation, and active user trends, helping admins optimize seat allocation and drive engagement across the app. Log in to your Loom workspace.
	From the admin tools menu, select Overview.
	Review the Overview, License allocation, and Most active users sections.
	Select Download
