## Description

With improved flexibility, space admins can now manage which types of work contributors can create and customize the create form template for each type of work. This reduces clutter and helps maintain an organized workflow in Jira Product Discovery spaces. To manage the types of work for contributors and configure the create form per type: Go to your Jira Product Discovert space.
	From the upper navigation bar, select +Create, then select Configure form.
	Under Allow contributors to create ideas
