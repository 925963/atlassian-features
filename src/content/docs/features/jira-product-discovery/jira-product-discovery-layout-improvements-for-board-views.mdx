## Description

The board view now offers more layout options to optimize the data displayed on the board cards in Jira Product Discovery. You can resize columns by changing the card width to small, default, or large to better accommodate their content and screen space. For the Premium plan, we’ve added additional control over how connection fields appear on board cards. You can select either a summary view of connections and their count or a detailed list view with all connections.
