## Description

Adding global fields to your Jira Product Discovery spaces is now faster and more intuitive. You can add global fields directly from any view to achieve consistency across spaces and reduce setup time. When creating new space fields, you can see suggestions for existing global fields you can reuse. To add a global field: In your Jira Product Discovery space, go to any view.
	In the right sidebar, select Fields > Add a global field.
	Select Discovery fields or Jira fields, or search by a field na
