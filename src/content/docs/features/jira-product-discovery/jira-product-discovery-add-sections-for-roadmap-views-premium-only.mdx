## Description

Organizing your roadmap views into logical sections can help you navigate and maintain your roadmap structure more easily. You can create sections to group related views, then drag and drop existing views into these sections, or add new views directly where needed. To create a section in the roadmap: Go to Jira Product Discovery.
	On the left sidebar, select Roadmaps > View all roadmaps.
