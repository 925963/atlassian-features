## Description

To improve visibility and tracking of work across your organization, you can now link work items across different Atlassian sites directly from the Overview tab in Jira Product Discovery. Once your admin sets up application links to remote Cloud sites, a new Site field appears when linking work items to ideas. To link a work item: In Jira Product Discovery, go to the target space and select the idea where you want to link a work item.
	In the Overview panel on the right, select Link work item.
