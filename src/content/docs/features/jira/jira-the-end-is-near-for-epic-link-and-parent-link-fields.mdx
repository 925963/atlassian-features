## Description

For a little over a year, we've been working to combine the Epic Link and Parent link fields in company-managed projects into the new universal Parent field (which you can read about here). To prevent data loss, the values from both fields have been automatically ported into the Parent field. With this change, the values in the Epic Link and Parent link fields are no longer visible on the Issue view.
