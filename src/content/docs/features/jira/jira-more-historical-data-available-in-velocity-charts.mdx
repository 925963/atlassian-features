## Description

Velocity charts in Jira now show data from more sprints, giving you a clearer picture of your team's productivity over a longer period of time. We've also added an average line so you can easily track your team's average output at a glance. Read more about the velocity chart To view the velocity chart: Start in your Jira software project.
	Select Reports.
	Select Velocity report if you're in a team-managed project, or Velocity chart in a company-managed project.
