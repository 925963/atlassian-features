## Description

Using the timeline in software projects, you can now add a assignee straightaway when creating a work item. To create a work item with an assignee: Select + Create Epic, or hover over a parent work item and select Create a child work item ( ).
	Give the work item a name.
	Choose a work type from the dropdown (if your project has more than one work type).
	Select the assignee icon and choose an assignee.
	Press enter or return on your keyboard to create your work item.
