## Description

Project summaries are now available across all Jira Cloud software projects! The _s_ummary view helps you monitor your project's progress by giving you a high-level overview of your team’s work. It includes a snapshot of your project's status, recent activity, priorities, team workload distribution, and epic progress. All Jira software projects have the summary view enabled by default. To view a project summary, navigate to your project, and then select Summary. Read more about the summary view
