## Description

If you need to export your Jira issues, you’ll notice a faster and smoother experience due to the following improvements: Speed and reliability — the export process is faster and can handle a large number of fields and extensive data sets
	Tracking and control — you can monitor the progress of your export, and cancel if needed To try it out, go to the Issues page in your project. Select Export, then select a CSV export option.
