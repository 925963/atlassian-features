## Description

Use AI to find and link similar work items or spot duplicates before they’re created. When you create a new work item, Jira uses AI to automatically find and surface work items with a similar summary. You can then link the existing work items to the one you’re creating. It helps you quickly build context, connect past work, and understand complexity from the start. Read more about Atlassian Intelligence features
