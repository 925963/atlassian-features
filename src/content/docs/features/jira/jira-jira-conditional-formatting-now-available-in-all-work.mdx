## Description

You can now apply and edit conditional formatting rules directly within All Work, not just in the list view. This allows for consistent formatting and easier management of your work items. To apply conditional formatting: Go to the All Work tab in your project.
	Select (…) More actions then Format rules.
	Edit or apply rules as needed.
