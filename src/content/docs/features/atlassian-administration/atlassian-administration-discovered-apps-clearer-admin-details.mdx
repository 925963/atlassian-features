## Description

To help you better manage your Discovered apps (Shadow IT), we now display the organization admin’s name and profile picture in the table. We’ve also added a summary of how the user count is calculated. How to review discovered apps To view details of discovered apps: Go to admin.atlassian.com. Select your organization if you have more than one.
	Select Security > Discovered apps.
