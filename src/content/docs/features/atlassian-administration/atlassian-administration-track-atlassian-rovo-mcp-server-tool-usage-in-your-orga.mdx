## Description

You can now track every time a tool is used through the Atlassian Rovo MCP server in your organization’s audit logs. This update gives admins more transparency and helps meet security and compliance requirements by recording each tool invocation event. To review Atlassian Rovo MCP server tool invocation logs in your audit logs: Go to admin.atlassian.com.
	Select your organization.
	In the left navigation, select Insights > Audit log.
	Filter for “Rovo MCP User Actions” to see tool invocation eve
