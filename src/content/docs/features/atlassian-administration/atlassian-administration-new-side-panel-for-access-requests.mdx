## Description

The access request experience in Atlassian Administration now uses either a modal or a new side panel, depending on your organization view. This update provides a more consistent and intuitive interface for managing user access requests, and makes it easier for admins to review and respond to requests. To see the new experience: Go to Atlassian Administration. Select your organization if you have more than one.
