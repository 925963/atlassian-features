## Description

Organization admins can now enable App Trust when creating or editing Microsoft Intune mobile app management (MAM) policies in Atlassian Administration. This change allows you to manage IP access requirements alongside your existing mobile security settings, providing more granular control over how managed users access Atlassian apps on mobile devices. To get started with this change: Log in to admin.atlassian.com.
	Select your organization, then go to Security and Mobile security.
	Choose Intun
