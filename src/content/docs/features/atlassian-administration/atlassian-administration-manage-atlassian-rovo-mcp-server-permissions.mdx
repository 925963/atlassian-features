## Description

Organization admins can now control Atlassian Rovo MCP server permissions across their organization using the new Permissions tab in Atlassian Administration. This change provides granular control over how the MCP server interacts with your data, allowing you to allow or block access by intent and app. To get started with this change: Go to Atlassian Administration, then Rovo.
