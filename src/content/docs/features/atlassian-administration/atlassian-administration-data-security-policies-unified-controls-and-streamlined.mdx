## Description

Admins can now manage organization-wide Data Security Policies in Atlassian Administration with a simplified experience. The new controls make it easier to configure, review, and enforce security policies across all Atlassian cloud products, reducing conflicts and simplifying policy management. To get started with this change: Go to Atlassian Administration.
	Select Security, then Data protection, and open the Data security policy page.
	Choose a control to view or edit its configuration.
