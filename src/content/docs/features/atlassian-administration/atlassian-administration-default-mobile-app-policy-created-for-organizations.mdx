## Description

Organizations with Atlassian Guard Standard and a verified domain will now automatically have a default mobile app policy created for them if one doesn't already exist. To view or manage your default mobile app policy: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > Device Security > Mobile app policies to review or adjust your default mobile app policy.
