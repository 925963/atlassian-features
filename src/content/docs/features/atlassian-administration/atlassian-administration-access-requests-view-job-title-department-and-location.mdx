## Description

Organization admins can now view a user's job title, department, and location directly within access requests in Atlassian Administration. This additional context helps admins make more informed decisions when approving or denying requests for product access, reducing the likelihood of requests being ignored due to a lack of information. To get started with this change: Go to Administration at admin.atlassian.com.
