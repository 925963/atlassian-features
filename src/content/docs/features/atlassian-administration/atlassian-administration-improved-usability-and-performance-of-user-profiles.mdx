## Description

We made user profiles easier to use. This change applies to customers who have the centralized user management experience. We moved the Apps and Groups tables into tabs. You can now choose how many apps or groups to show in each table, so you can see as many as you want at a glance. This update also adds filters to help you find certain apps or groups when there are many results. We made the user status more prominent so you can easily see if the user has access to your organization or not.
