## Description

Your existing System for Cross-Domain Identity Management (SCIM) API keys currently have an infinite lifespan. Starting on April 28, 2025, we will set your keys to expire between May 1, 2026 and May 1, 2027. What is an existing SCIM API key? An API key generated before January 1, 2025. Who does this change apply to? Identity providers that provision users with SCIM to an Atlassian organization. This change does not apply to these identity providers: Google Workspace or Microsoft Azure AD for nes
