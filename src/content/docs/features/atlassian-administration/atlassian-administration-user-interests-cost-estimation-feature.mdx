## Description

The User interests requests page in Atlassian Administration now includes a cost table that helps admins estimate costs. We’ve also included a link to compare costs for more detailed pricing. To get started with this change: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Apps > User requests.
	Select the User interests tab and open any trending interest.
	The request will open a side panel.
