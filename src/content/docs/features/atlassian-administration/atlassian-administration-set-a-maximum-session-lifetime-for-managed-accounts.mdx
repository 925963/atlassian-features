## Description

You can now set a maximum session lifetime for managed accounts. This lets you enforce a fixed session duration, regardless of user activity, to help meet your organization’s security and compliance requirements. To set a maximum session lifetime for managed accounts: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > User security > Authentication policies.
	Edit or create a policy to set a maximum session lifetime for managed accounts.
	Save
