## Description

Atlassian Administration now features updated charts powered by Atlassian Analytics. These charts offer improved performance, visual consistency, and enhanced accessibility, making it easier for admins to analyze data and uncover insights. To get started: Go to Atlassian Administration (admin.atlassian.com).
	Navigate to one of the following pages: Overview, Security Guide, or Platform Insights to view the updated charts.
