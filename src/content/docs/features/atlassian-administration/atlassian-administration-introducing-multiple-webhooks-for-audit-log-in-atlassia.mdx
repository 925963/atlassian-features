## Description

We’re excited to announce a major enhancement to audit log integrations in Atlassian Administration, you can now register up to three webhooks per organization! To register a webhook URL: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > Audit log.
	Select Settings.
	Select the Webhook tab.
	Select Add webhook.
	Enter a Name for the webhook.
	Enter the Webhook URL (the destination where you wish to receive real-time activity updates).
