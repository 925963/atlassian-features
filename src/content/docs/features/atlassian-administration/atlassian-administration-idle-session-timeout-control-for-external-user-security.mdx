## Description

A new idle session timeout control is now available in external user security policies. This feature helps admins manage session security for external users across Atlassian apps, reducing the risk of unauthorized access due to inactive sessions. To get started with this change: Go to Atlassian Administration.
	Select Security, then External user security policies.
	Configure the idle session timeout control as needed for your organization.
