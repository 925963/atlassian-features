## Description

The site-level access requests page is being retired. Now, when site admins need to manage access requests, they will be directed to the User request management page for a more streamlined experience. To manage access requests as a site admin: Go to Atlassian Administration.
	Select User requests from the menu.
	Review and manage all pending access requests from this central location.
