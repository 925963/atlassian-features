## Description

You can now select Rovo as a mobile app in all Mobile Application Management (MAM) security policies, including both regular and Intune policies, regardless of any other settings in place. To include Rovo in your mobile security policies: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > Device security > Mobile app policies.
	Select Rovo from the list of available mobile apps.
	Apply or update your policy as needed.
