## Description

Admins can now apply IP allowlisting controls to Rovo experiences at the organization level in Admin hub. This helps ensure that only users from approved IP addresses can access Rovo Search, Chat, Bookmarks, and more, improving security for your organization. To learn more, see Specify IP addresses for app access. Go to Atlassian administration (admin.atlassian.com) and select your organization.
	Select Security > Device Security > IP allowlisting
	Create allowlist or Edit an existing allowlist
