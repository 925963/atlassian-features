## Description

Assets in Jira Service Management is now included in the backup and restore of your Atlassian Cloud services. Make sure you have the required permissions to use backup and restore features in your JSM site.
	Go to the backup and restore section in your Atlassian Cloud admin settings.
	Follow the steps to create a backup or restore your JSM Assets data.
