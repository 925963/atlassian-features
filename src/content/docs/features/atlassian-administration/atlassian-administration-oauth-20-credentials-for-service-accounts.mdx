## Description

We’ve added a new authentication option for service accounts. Now, you can find options for both OAuth 2-Legged (2LO) authentication and API Tokens combined under a single menu making it easier to manage your service accounts authentication. To use the new combined menu for service account credentials: Go to the Admin Hub in your Atlassian app.
