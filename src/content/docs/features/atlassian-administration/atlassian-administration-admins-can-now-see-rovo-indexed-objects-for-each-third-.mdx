## Description

Organization admins can now see the number of indexed objects against each Rovo third-party connector. To find out more about indexed objects and Rovo quotas, see Rovo usage quotas. To see the object count: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Settings > Rovo.
	Expand the site where you’ve connected a third-party app.
