## Description

We’re going to remove the feature that lets admins suggest changes to an external user’s details from the user’s profile. This feature is only available in the centralized user management experience. We know this feature doesn’t help admins directly update an external user’s details, which can only be updated by the user themselves. As a result, we’ve consistently seen low usage.
