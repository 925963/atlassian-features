## Description

Some Marketplace apps can now connect to multiple Atlassian apps. If a Marketplace app supports this, you can install it once and choose which Atlassian apps—like Jira, Confluence, and Compass—you want to connect it to on installation. You can update these connections at any time in Atlassian Administration. During installation, you can choose which Atlassian apps to connect to, and you can update these connections later in the Atlassian Administration Connected apps screen. For Marketplace apps
