## Description

We’ve updated the Insights page so there’s less scrolling to find the right user and activity charts. There are now tabs on the page to group related charts. To view your organization’s refreshed Insights page: Go to admin.atlassian.com. Select your organization if you have more than one.
	Select Security > Insights.
	Select a tab to see your charts.
