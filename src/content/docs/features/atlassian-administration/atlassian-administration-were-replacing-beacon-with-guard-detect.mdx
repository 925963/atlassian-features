## Description

This update only applies to organizations that participated in the Beacon or Atlassian Information Security beta. Beacon (beta) will soon be part of Atlassian Guard Premium. Read the blog We’re replacing ‘Beacon’ with ‘Guard Detect’ in the CSV files you export. The name of your groups (such as beacon-admins) will not change.
