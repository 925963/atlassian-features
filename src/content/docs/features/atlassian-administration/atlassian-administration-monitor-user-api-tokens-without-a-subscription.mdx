## Description

You no longer need an Atlassian Guard Standard subscription to monitor the usage of user API tokens. At no extra cost, you can view when a token expires, when it was last used and who created it. You can also revoke a token if you need to. You can track these details about user API tokens only for your managed accounts. Explore user API tokens
