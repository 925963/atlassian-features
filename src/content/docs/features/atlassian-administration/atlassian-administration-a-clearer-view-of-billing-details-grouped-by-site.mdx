## Description

We have updated the billing preview table in Atlassian Admin to group billing details by site and load entitlements only when you need them. This makes it easier to review and manage billing information for each site. To view and use the new grouped billing preview table in Atlassian Admin: Go to Atlassian Admin in your Atlassian app.
	Select Billing from the navigation menu.
	Find the Billing preview section to see billing details grouped by site.
	Select a site to load and review its entitleme
