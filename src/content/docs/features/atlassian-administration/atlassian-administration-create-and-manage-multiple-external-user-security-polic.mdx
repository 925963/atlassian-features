## Description

You can now create and manage external user security policies in Atlassian Administration. This lets you set different security rules for groups of external users by email domain. To create and manage multiple external user security policies: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > User security > External users.
	Select the Create new policy button to add a new policy.
