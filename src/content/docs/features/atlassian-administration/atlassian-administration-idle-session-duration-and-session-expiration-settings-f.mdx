## Description

Administrators can now configure Idle session duration and Session expiration in authentication policies for Atlassian Government Cloud environments using Atlassian Administration. Previously, these settings were fixed at 24 hours. This update helps organizations meet their government compliance requirements by allowing more control over session security. To get started with this change: Go to Atlassian Administration at admin.atlassian.com.
	Select your organization.
	Navigate to Security, User
