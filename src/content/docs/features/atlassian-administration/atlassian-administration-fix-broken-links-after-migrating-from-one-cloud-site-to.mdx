## Description

When migrating Jira Cloud or Confluence Cloud to a different site, some links between individual entities may break due to changes in the URL. This typically includes: Links to Jira work items that users have pasted as URLs in comments and descriptions. Links from other instances to the migrated site, such as links from Confluence pages to your Jira work items.
