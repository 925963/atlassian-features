## Description

Atlassian Backup and Restore now supports data residency. When an app is pinned to a data residency region, all new backups (including product data, media, and user and group data) are stored in the pinned region. Admins can view the storage region for each backup in the backup details panel. Restore operations enforce region matching: the destination app must be unpinned or in the same region as the backup.
