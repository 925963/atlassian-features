## Description

We've enhanced the webhook registration process to simplify how you edit webhook details. To edit the details of a registered webhook: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > Audit log.
	Select Settings.
	Select the Webhook tab.
	Select Edit details to update the details.
