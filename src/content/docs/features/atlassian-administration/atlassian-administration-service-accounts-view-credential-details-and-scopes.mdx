## Description

Organization admins can now view the specific details and scopes of each credential tied to a service account in Atlassian Administration. This update provides a side panel that displays information for both API tokens and OAuth credentials, helping you stay aware of the permissions attached to each credential. To get started with this change: Go to Atlassian Administration (admin.atlassian.com).
