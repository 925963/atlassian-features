## Description

We’re separating this Admin API key scope: API keys and tokens. You could only choose the API keys and tokens scope together, but now you can choose one or the other. We want to give you more control when you select an Admin API key scope. What do you need to do?When we make this change, we automatically assign the scope to API tokens only.
