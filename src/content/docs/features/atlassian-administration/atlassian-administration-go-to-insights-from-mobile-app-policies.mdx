## Description

We added a link to mobile usage insights on the mobile app policies page. You can now navigate to mobile insights easily after creating or updating a mobile app policy. To view mobile app usage information: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Security > Insights.
	Select the Mobile usage tab.
