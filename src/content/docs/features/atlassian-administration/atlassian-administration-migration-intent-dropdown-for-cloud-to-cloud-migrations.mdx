## Description

A new migration intent dropdown has been added to the cloud-to-cloud migration experience in Atlassian Administration. Admins can now specify whether a migration is a test or production migration. This intent is used by Atlassian’s Support and on-call teams to correctly assess incident severity and respond faster when production migrations fail, reducing manual back-and-forth. Go to Atlassian Administration. Select your organization if you have more than one.
