## Description

You can now use Audit Log Query Language (ALQL) to search and filter audit logs with advanced, flexible queries—similar to Jira Query Language (JQL). This makes it easier to find the information you need, even as your organization’s audit data grows. To use ALQL in the Audit Log Platform: Go to the Audit Log section in Atlassian Administration (admin.atlassian.com).
	Select the option to switch from basic filters to advanced search.
	Enter your query using ALQL syntax in the advanced search bar.
