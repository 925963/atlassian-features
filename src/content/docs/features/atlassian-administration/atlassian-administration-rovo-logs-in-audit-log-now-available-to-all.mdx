## Description

Rovo logs in Audit log within Atlassian Administration are now accessible to everyone, including those on the free plan. Previously, access to Audit log in Atlassian Administration was gated by an Atlassian Guard Standard subscription. This meant that only customers with Guard Standard or higher could view audit events, including those generated by Rovo.
