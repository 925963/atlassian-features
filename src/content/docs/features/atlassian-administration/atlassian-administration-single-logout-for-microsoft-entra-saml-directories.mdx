## Description

Administrators can now enable service provider-initiated SAML single logout for Microsoft Entra directories in Atlassian Administration. This ensures that when users log out of Atlassian, their Microsoft Entra SAML session is also terminated, closing a security gap and improving account protection for organizations using Microsoft Entra as their identity provider. To get started with this change: Go to Atlassian Administration and select your organization.Navigate to Security, then User Security
