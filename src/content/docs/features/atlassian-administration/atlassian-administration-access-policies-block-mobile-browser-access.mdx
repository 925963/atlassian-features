## Description

Atlassian Guard now allows organization admins to block access to Atlassian apps from mobile browsers. This new condition-based policy helps you enhance security by ensuring users only access organization content through approved methods, such as Atlassian mobile apps or desktop browsers. To get started with this change: Go to admin.atlassian.com. Select your organization if you have more than one.
