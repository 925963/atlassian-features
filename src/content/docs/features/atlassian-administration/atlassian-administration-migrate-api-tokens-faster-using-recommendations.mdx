## Description

We've introduced a new Token type column in the API token activity page to help you identify API tokens. This allows you to see their details and access recommendations when migrating tokens. To see token details and recommendations: Go to Atlassian Administration. Select your organization if you have more than one.
	Select Insights > API token activity.
	Select a token from the Token type column to view migration recommendations.
