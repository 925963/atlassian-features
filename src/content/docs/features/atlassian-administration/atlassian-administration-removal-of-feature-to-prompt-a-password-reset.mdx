## Description

We’re going to remove the feature that lets admins prompt an external user to reset their password. We recognize the inability to force a password reset and confirm that a user has changed their password means this feature hasn’t been an effective way to secure an external user’s account. We’ll begin removing this feature in November 2025. We recommend admins contact their external users through email or other channels instead when you need to ask them to reset their password.
