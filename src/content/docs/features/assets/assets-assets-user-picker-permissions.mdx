## Description

Assets now has a dedicated permission model that replaces the reliance on Jira global permissions. Assets admins can now directly manage who can find people and groups in user pickers through a new global permissions interface. To get started with this change: Go to Assets, then select the Settings icon.
	Select Assets settings, then Global permissions.
	Select the User picker.
	Choose All Assets users, All Assets admins, or specific groups to grant access.
