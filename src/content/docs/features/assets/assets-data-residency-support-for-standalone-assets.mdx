## Description

Administrators can now manage data residency for the standalone Assets app in Atlassian Administration. When you move any Jira family app, Assets will move with it, giving you more control over where your data is stored. (Data residency can only start with Jira.) To get started with this change: Go to Atlassian Administration.
	Select Security and then Data residency.
	View or move the Assets app alongside your Jira family apps.
