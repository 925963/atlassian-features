## Description

When new users are added to a Confluence site without a lot of content, they will now land directly in the editor with relevant template recommendations. This helps new users start collaborating faster and more effectively in Confluence. If your site has fewer than 10 users: Add a new user to your Confluence site.
	The new user will be directed to the editor with template suggestions to help them begin working right away.
