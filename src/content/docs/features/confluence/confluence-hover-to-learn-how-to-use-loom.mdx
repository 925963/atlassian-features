## Description

Confluence users can now see a spotlight when hovering over the Loom button in the floating toolbar or inline comments. This brief pop-up explains what Loom does and how to use it, helping you quickly understand how to share your work and add context with video. This change applies to Confluence sites where Loom was auto-provisioned for you. To get started with this change: Go to any Confluence page.
