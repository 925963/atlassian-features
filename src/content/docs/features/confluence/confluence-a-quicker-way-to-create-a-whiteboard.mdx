## Description

We have updated the order of options in the create menu so that creating a whiteboard is now easier to find and faster to access. To explore the new create menu order: Open Confluence.
	Select the Create button from the main navigation.
	Look for the Whiteboard option, which now appears higher in the list.
	Select Whiteboard to create your new whiteboard.
