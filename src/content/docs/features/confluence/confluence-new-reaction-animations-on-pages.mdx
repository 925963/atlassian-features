## Description

When you return to a page in Confluence that has received new reactions since your last visit, you'll now see a fun animation highlighting those reactions. This update makes it easier to stay engaged with your team's feedback and quickly identify which content is resonating with others. To get started with this change Open a Confluence page that you have previously visited.
	Look at the reactions section of the page.
	Observe the animation if new reactions have been added since your last visit.
