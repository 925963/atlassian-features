## Description

We’re making sharing in Confluence more intuitive with a new live share experience. Changes you make to permissions now apply immediately without needing to select a save button, ensuring the dialog always reflects the current source of truth. Additionally, anyone you explicitly share with now stays on the specific access list even if you change the general access level, giving you peace of mind that your individual permissions remain exactly as you set them.
