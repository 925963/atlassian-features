## Description

We've expanded the selection of arrowheads when creating diagrams in whiteboards, giving you more ways to visualize connections and flows. This improvement also allows you to create advanced and expressive diagrams. To use the new arrow in your whiteboard: Create a blank whiteboard and insert a line.
	Select the line to open the toolbar.
	Select the arrow dropdown and choose from the expanded range of arrowheads to create your diagram.
