## Description

As you drag elements near three or more aligned items, visual guides appear to help you quickly space them evenly. This helps your whiteboard stay clean and consistent. To evenly space the elements in your whiteboard: Select one or more elements such as stickies, frames, or shapes.
	Drag them near three or more aligned items so they’re close to evenly spaced.
	When you see the visual guides appear, drop the element into place.
