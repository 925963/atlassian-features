## Description

We've upgraded the Create menu in Confluence to help you find what you need faster. The new experience separates how you create from what you create, making it easier to start a new page or record a Loom video directly from the top navigation. To get started with this change: Select Create from the top navigation.
	Choose the type of content you want to create, such as a page or a Loom video.
	If you have many creation options, select More to see the full list in the overflow menu.
