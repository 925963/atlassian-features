## Description

This change introduces new automation triggers in Confluence that let you create automation rules when a user is added to or removed from a team. These triggers help teams streamline processes and keep information up to date automatically. Go to the automation settings in Confluence.
	Create a new automation rule and select the trigger for when a user is added to or removed from a team.
	Configure the actions you want to automate and save your rule.
