## Description

Confluence and Jira now display in-app notifications encouraging users to connect third-party providers when they interact with unsupported smart links. This helps users unlock richer features and a more comprehensive experience by integrating external data sources. Paste a Smart Link from a third-party provider (such as Google Drive) in Confluence or Jira.
	If you have not yet connected the provider, an in-app notification will prompt you to authenticate and connect your account.
