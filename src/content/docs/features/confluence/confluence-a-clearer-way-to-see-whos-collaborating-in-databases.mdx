## Description

This update adds visual highlights to cells and cards in databases when multiple people are working together, making it easier to see where others are collaborating in real time. To see the new collaboration highlights in databases: Open any database content type in Confluence.
	Invite or join others in editing the same database.
	Look for highlighted cells or cards that show where your collaborators are working.
