## Description

You can now search for people using email addresses when sharing Confluence content. This makes it easier to find the right people when multiple users have the same name or if you only know someone’s email address. Open any Confluence content you want to share.
	Select the Share button at the top right.
	In the Add names, groups, or emails search field, enter an email address.
	Select the person you want to share with from the entry that appears.
