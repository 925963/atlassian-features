## Description

We’re introducing IP allowlist restrictions for media asset domains. Only IP addresses on your organization’s allowlist will be able to access media assets (attachments, images, and videos) hosted on media-cdn.atlassian.com and api.media.atlassian.com. If you use third-party security tools like Zscaler, make sure to allowlist [Source IP Anchoring|https://help.zscaler.com/zia/understanding-source-ip-anchoring] *.atlassian.com to avoid disruptions.
