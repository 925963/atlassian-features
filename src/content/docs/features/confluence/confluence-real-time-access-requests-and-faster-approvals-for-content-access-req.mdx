## Description

We’re introducing real-time access requests in Confluence. If you can manage access for a content, you’ll now be able to quickly approve or deny requests if you’re active on the content through a flag, without interrupting your workflow. To respond to content access requests in real time: Open any Confluence content type (such as a published page, live doc, or whiteboard) that you can manage permissions for.
