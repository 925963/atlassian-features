## Description

You can now attach multiple line labels to a path and drag labels off paths in Confluence whiteboards. This update gives you more flexibility to organize and clarify your diagrams, making it easier to communicate complex ideas visually. Open a whiteboard in Confluence.
	Select a path on your diagram.
	Add or edit line labels as needed. You can now attach more than one label to the same path and drag existing labels off paths.
