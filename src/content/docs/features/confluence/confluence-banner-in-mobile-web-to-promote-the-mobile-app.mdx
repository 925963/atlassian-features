## Description

Confluence mobile web users will now see a banner encouraging them to try the Confluence mobile app. This change aims to help users discover the benefits of the app, such as faster access and richer features, leading to a better experience on the go. To get started with this change: Open Confluence in your mobile web browser.
	Look for the new banner promoting the Confluence mobile app.
	Select the Open app option.
