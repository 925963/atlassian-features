## Description

If the content tree fails to load in Confluence, you'll now see an updated error view with a Retry button. This improvement makes it easier to refresh the navigation sidebar without reloading the entire page. To get started with this change: Navigate to any Confluence space.
	If the sidebar fails to load, locate the error message in the content tree area.
	Select Retry to attempt to load your pages and folders again.
