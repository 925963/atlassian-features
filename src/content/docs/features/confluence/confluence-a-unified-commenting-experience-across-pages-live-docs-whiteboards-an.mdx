## Description

We’ve introduced a unified comments panel across pages, live docs, whiteboards, and databases in Confluence. This provides a consistent experience that makes it easier for teams to collaborate and manage feedback, no matter where they work. Open a page, live doc, whiteboard, or database in Confluence.
	Select the comments icon on the action bar at the bottom right of the screen to open the new comments panel.
	Use the panel to view, reply, resolve, and manage comment threads.
