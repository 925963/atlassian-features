## Description

Now, anyone can invite someone new or request access for them in Confluence and set their content permissions – all in one step. To invite someone and set their access: Open a content type in Confluence.
	Select Share.
	Enter the email address of the person you want to invite or request access for.
	Choose the intended content permissions for them.
	Select Share to complete the process.
