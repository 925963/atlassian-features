## Description

We’ve updated the onboarding that appears for new Confluence users. It now has a more modern look, and the content has been rewritten for clarity. These changes help new team members better understand what to do next when getting started. To see the updated onboarding experience in Confluence: Sign in to Confluence as a new user.
	Navigate to any Confluence content type.
	Wait for the onboarding content to appear after a short delay.
