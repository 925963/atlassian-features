## Description

We’ve updated the icon used to open the comment panel in Confluence. The new icon makes it easier to understand how to open and close comments on your content. To get started with this change: Open any Confluence content type, such as a published page or blog post.
	Find the inline comment panel icon in the inline comment component.
	Select the icon to open or close the comment panel as usual.
