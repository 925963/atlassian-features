## Description

Confluence now includes a persistent toolbar at the bottom of the screen, keeping page reactions and comments always visible and easy to access while viewing content. This helps people collaborate more easily by reducing friction for adding reactions or comments. To get started with this change: Open any Confluence page where you have permission to react or comment.
	Use the toolbar at the bottom of the screen to add a reaction or comment.
