## Description

When you're finished reading a page on Confluence, you'll now see a flag in the bottom-left corner that will let you react to your teammate's page quickly and easily, so you can let them know you appreciate their hard work. This feature is now being released to more users, so even more teams can enjoy this easy way to engage! To try out this new way to react: Open any published content in Confluence.
