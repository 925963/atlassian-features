## Description

Confluence page analytics now feature inbound and outbound link click analytics. Inbound link analytics show pages that link to the current page and how often users arrive through those links. Outbound link analytics show click counts and click-through rates for links included on the page. Analytics data includes Confluence active pages and blogs after Jan 31, 2025.
