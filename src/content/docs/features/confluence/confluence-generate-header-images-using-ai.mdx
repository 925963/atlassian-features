## Description

Polish your Confluence pages faster and make them more visually engaging with AI-generated header images. Confluence can now suggest a relevant image based on your page content or propose a high-quality prompt to help you produce the perfect header. To get started with this change: Open a page that does not have a header image.
	Select the header image entry point to open the generation dialog.
	Enter your own description or click the “Suggest” button to be recommend a description
	Select Genera
