## Description

Table scaling features are now available in Confluence, Jira, and Bitbucket. Tables automatically scale to fit narrower viewports. These updates make working with tables faster and more intuitive across Atlassian apps. To get started with this change: Open a table in the editor of Confluence, Jira, or Bitbucket.
	Resize your browser window to see tables scale automatically.
