## Description

There are now more ways you can search for people using email addresses. This makes it easier to find the right people when multiple users have the same name or if you only know someone's email address. Any Confluence surface with a user search field has this updated behavior, including share, change owner, content manager, and more. Go to any user search field in Confluence, such as share, change owner, or content manager.
