## Description

The live docs toggle is now enabled by default when creating a knowledge base space in Confluence. Select Create in the top navigation, then choose Space.
	
		Or, open the Spaces dropdown, select View all spaces, then choose Create a space.
	
	
	When you select the knowledge base space type, the live docs toggle will be enabled by default.
	After creating the space, you can access live docs from the Create menu in the left navigation.
