## Description

You can now access over 1,700 new cloud architecture icons from Amazon Web Services (AWS), Microsoft Azure, and Google Cloud Platform (GCP) in Confluence Whiteboards. These icons are available to Premium and Enterprise customers, making it easier to create clear and professional architecture diagrams. To find and use the new cloud architecture icons in Confluence Whiteboards: Open or create a Whiteboard in Confluence.
	Select the shape icon in the bottom or main toolbar.
	Select the More shapes
