## Description

Create a native embedded database directly from your whiteboard by selecting + (more) on the toolbar menu. This makes it easier to add and use databases within your whiteboards, helping you organize and manage information more efficiently. Open a Confluence Whiteboard.
	Select + (more) on the toolbar menu.
	Select Create database to add a new embedded database to your whiteboard.
