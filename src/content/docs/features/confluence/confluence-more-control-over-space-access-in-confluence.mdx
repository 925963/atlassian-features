## Description

We’re introducing more granular space permissions to support role-based access control in Confluence. If your site is using roles, you can now delegate tasks like managing templates, space look and feel, and space content — without giving full admin access. We’re also separating the ability to delete your own comments into its own permission, giving admins more control over who can remove comments across spaces. To use these new permissions, first check that your site is using roles.
