## Description

We're updating how Expand elements work for classic pages in Confluence. These improvements ensure your view of expanded or collapsed content is independent and does not affect other users on the same page. To get started with this change Open any classic page in your Confluence app.
	Select Edit to begin editing.
	Insert an Expand element into the page.
