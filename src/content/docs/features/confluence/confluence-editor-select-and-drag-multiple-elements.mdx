## Description

We're introducing a new feature to the Confluence Cloud and platform editor that lets you drag and drop multiple selected elements. This update allows you to move multiple elements together using a drag handle. What's new Multi-select drag and drop: You can now select and move multiple elements simultaneously by using the drag handle.
