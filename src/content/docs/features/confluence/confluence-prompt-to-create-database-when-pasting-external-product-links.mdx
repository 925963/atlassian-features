## Description

When you paste a link to an external product such as Google Sheets, Smartsheet, or Airtable into a Confluence page, live edit page, or blog, Confluence will prompt you to use the native database feature. This helps you organize and manage your data more effectively within Confluence. To get started with this change: Open a Confluence page, live edit page, or blog post.
	Paste a link to an external product that resolves to a valid Smart Link (such as Google Sheets, Smartsheet, or Airtable).
