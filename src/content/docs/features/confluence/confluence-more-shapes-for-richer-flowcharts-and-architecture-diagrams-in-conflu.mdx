## Description

Now you can create more detailed flowcharts and lightweight architecture diagrams in Confluence Whiteboards. Access a wider variety of shapes directly from the editing toolbar, making it easier to represent complex processes and technical systems. To use the new shapes in Confluence Whiteboards: Select the shape in the toolbar at the bottom of your whiteboard.
