## Description

The Create diagram (whiteboard) and Create database options are now surfaced at the top of the slash (/) menu (just below Ask Rovo), improving visibility and reducing the need to scroll or search. Open or start editing a Confluence page or live doc.
	Type / to open the insert menu.
	Select Create diagram or Create database.
	Your new whiteboard or database will open in a new tab and automatically be inserted into your page or live doc.
