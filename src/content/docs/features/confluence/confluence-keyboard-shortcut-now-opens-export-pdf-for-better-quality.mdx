## Description

Confluence now guides you to use the Export PDF feature for the best-quality PDFs. The ⌘P/Ctrl+P shortcut now opens Export PDF instead of browser print, giving you higher-quality PDFs. If you use your browser's print menu, you'll see a guide to Export PDF, but you can still print through your browser if needed. To create a high-quality PDF from your Confluence page or live doc: Select ⌘P/Ctrl+P.
	Confluence will open the Export PDF feature.
	Download your PDF.
