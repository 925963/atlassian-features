## Description

Now you can include an optional message in notification emails that are sent to participants when creating or updating an event in team calendar. This new field lets you send context-specific information directly to the people you invite. Open your team calendar.
	Create or update an event.
	Look for the new optional field under ‘Message’ in the event creation/update window.
	Enter your message and complete the event as usual.
