## Description

When you create a page in Confluence using the Create with Rovo flow, an relevant header image is now automatically generated and applied based on your content. This update helps you save time on page formatting by providing high-quality, context-aware visuals with zero manual prompting. Select Create in the top navigation bar of Confluence.
	Select Create with Rovo.
	Enter your prompt and generate your page content.
	Observe the automatically applied header image at the top of your new page.
