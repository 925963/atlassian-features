## Description

We've redesigned the Questions interface so it's easier to read and respond to questions. Some elements that've been refreshed include the vote button as well as the Questions and Expert tab. This change applies to both the Confluence Questions app and Questions macro. To see the redesigned interface, select Questions under Apps.
