## Description

Improvements to Confluence whiteboards include the ability to: attach lines to the same shape and snapping lines to shapes to improve precision and complexity in diagrams - drag sections from their title, smart sections won’t action on every load - bulk import all Jira issues from a JQL query - click on the zoom percentage to easily adjust zoom level
