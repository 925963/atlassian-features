## Description

We're introducing a new feature that automatically generates a page title using AI when you attempt to publish a page without one. This update reduces the need for inserting a title before publishing a page. What's new: AI-generated titles on publish: Automatically suggests a title when you publish without one.
	User confirmation required: You can accept or edit the AI-generated title before finalizing.
