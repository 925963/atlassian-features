## Description

We're changing the default behavior for image insertion in the editor to always insert images as blocks rather than inline. This update addresses user feedback and analytics showing a preference for block images, enhancing document layout and usability. Default block image insertion: Images will now be inserted as blocks by default, improving document structure.
