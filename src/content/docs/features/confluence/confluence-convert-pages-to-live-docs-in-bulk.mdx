## Description

Space admins can now use content manager to convert pages to live docs. Live docs allow collaborative real-time editing without the need to publish. In the left navigation bar, select the more actions menu in the space's name, then select Space settings.
	Select Content manager.
	A list of all space content displays. Select one or more pages.
