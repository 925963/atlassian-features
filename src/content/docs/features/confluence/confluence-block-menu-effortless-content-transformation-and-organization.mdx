## Description

The Block Menu in Confluence helps you quickly transform, move, and share content by opening a context menu from any block’s drag handle. This menu streamlines structural changes and collaboration, making it easier to manage content across paragraphs, lists, and more—without interrupting your workflow. To get started with this change: Open a page in the Confluence full page editor.
