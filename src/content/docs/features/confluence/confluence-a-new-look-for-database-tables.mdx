## Description

We have redesigned the database table interface with a modern look and we're introducing new features to streamline how you interact with data. This update includes enhanced selection options and a new action bar for more efficient editing and management. Add a database to Confluence to check out the new changes, including: Redesigned Table UI: The database table now features updated styling, clearer headers, and improved visual hierarchy for easier navigation and readability.
