## Description

The Create with Rovo feature now allows you to create and edit databases using natural language prompts, making it easier to quickly generate and manage structured information within your Confluence workspace. Open Confluence and navigate to Create, then Create with Rovo.
	Ask Rovo to create a database by submitting a prompt using natural language. It could contain your recent projects or Jira work items.
	Edit your database once Rovo has created it, and add it to your Confluence space.
