## Description

Confluence now includes a more convenient way to build your own AI assistants with the addition of a new action in the creation menus. This change allows you to select Create agent directly from both the global and contextual create buttons, making it faster to start building agents that help automate your workflows and answer questions. To get started with this change: Go to the global navigation bar at the top of Confluence.
