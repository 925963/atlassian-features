## Description

You can now see Forge (confluence:contextMenu) and Connect (page.view.selection/action-panel) extension points in the Confluence cloud editor toolbar. This update makes it easier to access and use your favorite apps while working on your content. Open the Confluence cloud editor.
	Highlight some content in the editor.
	If your site has toolbar apps installed, you will see them displayed in your editor toolbar for quick access.
