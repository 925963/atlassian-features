## Description

You can now edit the Blog Posts macro directly in the Confluence Editor's side panel, instead of using the previous pop-up modals. This update makes it easier to manage macro settings and preview changes while editing your Confluence pages. To explore this change: Open a page in the Confluence Editor.
	Insert a Blog Posts macro onto your page.
	Edit the macro settings in the side panel that appears.
