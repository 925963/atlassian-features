## Description

We’ve added Summarize and Ask Rovo buttons to the Smart Links toolbar in the Confluence editor. This update helps you quickly understand the context of linked content from Atlassian and third-party apps with a single click, helping you stay focused without leaving your page. To get started with this change: Select any Smart Link within the editor.
	Select the Summarize button from the toolbar.
	Review the summary of the linked content.
