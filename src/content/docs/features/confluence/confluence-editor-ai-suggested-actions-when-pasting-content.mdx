## Description

We've replaced the floating toolbar in the Atlassian editor in Confluence with a new menu that appears when you paste content. This menu uses Atlassian AI to suggest relevant actions based on the content you've pasted, helping you format and organize your work more efficiently across Atlassian apps. To get started with this change: Open any Confluence.
	Paste text or content into the editor.
	Select the new menu that appears to view and apply AI-suggested actions.
