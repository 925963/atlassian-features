## Description

We are introducing support for nested tables in Confluence to address a key limitation in the cloud editor. You can now insert tables within tables up to one-level deep. This aims to assist migrations from the legacy editor to cloud, and improve content creation. What's new: Nested table insertion: In the cloud editor you can now insert tables within tables, up to one level deep.
	Migration support: This feature aids in migrating content from the legacy editor, allowing pages with nested tables
