## Description

When Confluence users with advanced search enabled dwell on the search results page for 15 or 20 seconds without finding what they need, the chat sidebar now opens automatically with query-specific conversation starters. This helps users get timely assistance and continue their search journey more effectively. To get started with this change: Begin a search in Confluence.
