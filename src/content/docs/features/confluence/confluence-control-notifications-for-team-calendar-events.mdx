## Description

Now you can decide whether to send notifications to watchers when creating or updating events in your Confluence team calendar. This new checkbox option gives you control over notifications, enhancing your ability to manage communication effectively. To use this new feature: Open your team calendar.
	Create or update an event.
	Look for the new checkbox option in the event creation/update window.
	Select the checkbox if you want to send notifications to watchers.
