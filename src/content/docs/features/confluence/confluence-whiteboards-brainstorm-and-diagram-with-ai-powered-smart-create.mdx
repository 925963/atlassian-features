## Description

You can now use AI-powered Smart Create in Confluence whiteboards to instantly generate brainstorming cards, flowcharts, and mind maps from a simple prompt. Ensure AI is turned on for the instance.
	Open a whiteboard.
	Select the Smart Create button from the right hand toolbar.
