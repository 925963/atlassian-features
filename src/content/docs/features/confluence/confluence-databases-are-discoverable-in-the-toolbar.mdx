## Description

We've made the database option discoverable in the toolbar to allow users to quickly insert databases, organize structured work in one place, and ensure information stays up to date across their pages. To insert a database using the toolbar: Open a Confluence page or a live doc.
	Select Insert elements then Database from the toolbar.
