## Description

The new version history experience in Confluence offers a modern interface for viewing, comparing, and managing page versions. Users can easily review previous versions, compare changes, delete or restore versions - and, if Rovo is enabled - ask AI to summarize version history. This update streamlines content management and enhances collaboration in Confluence. To get started with this change: Create or navigate to a Confluence page, blog, or live doc.
