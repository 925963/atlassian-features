## Description

This update improves filtering in Confluence Databases by enabling the right filter comparison options and value selectors for Jira fields, based on each field’s data type. This helps users find and organize information more efficiently across Confluence and Jira. To get started with this change: Open a Confluence Database that includes Jira fields.
	Select the filter option for a Jira field.
	Choose from the available comparison options and value selectors tailored to the field’s data type.
