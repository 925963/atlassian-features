## Description

When you search across Jira or Confluence, you will see the correct icon for each content type in the search dropdown results. This means you can quickly spot whether something is a page, live page, space, blog post, comment, or another type of content. To see the new icons in action: Open Jira or Confluence.
	Select the search bar.
	Begin typing your search term.
	Look at the icons next to each search result to identify the content type.
