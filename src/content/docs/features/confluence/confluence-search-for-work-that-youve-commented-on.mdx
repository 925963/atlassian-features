## Description

A new Commented by me filter is now available when you search for work. This filter helps you quickly find items you have commented on, making it easier to revisit your work and conversations. To get started with this change: Go to the search page.
	Select the filter dropdown, and select Jira.
	Select the Commented by me filter to find work you have commented on.
