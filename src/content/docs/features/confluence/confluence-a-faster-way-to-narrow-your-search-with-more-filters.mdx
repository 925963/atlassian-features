## Description

You can now use more filters in the search field to help you narrow your search results. This update introduces additional filtering options, making it easier to find the content you need. To use the new filters in Confluence's main search field: Open Confluence.
	Select the main search field from the top navigation bar.
	Use the new filter options that appear to refine your search results.
