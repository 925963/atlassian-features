## Description

We've added an opt-out functionality for the Audio briefing feature in Confluence, so admins can choose to opt-out of it if they prefer. You can do this by turning off the Beta features toggle in Atlassian Administration. We've also added a Labs lozenge next to the Audio briefing feature in Confluence.
