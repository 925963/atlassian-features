## Description

When you view pages on the web using your mobile, you'll see a prompt to discover the mobile app. With the app, you can catch-up on missed content using AI-powered audio features and quickly comment on pages your team members are working on. To download the Confluence mobile app, select Open in the banner on the page you're viewing in the mobile web.
