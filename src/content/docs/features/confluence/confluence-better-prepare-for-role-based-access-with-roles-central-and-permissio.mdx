## Description

In order to help you plan for Confluence roles, we’re introducing two new resources for Confluence admins — permissions data exports and Roles Central. Roles Central is your one stop shop for your transition to managing space access with roles instead of individual permissions. You’ll find all info and tools you’ll need there.
