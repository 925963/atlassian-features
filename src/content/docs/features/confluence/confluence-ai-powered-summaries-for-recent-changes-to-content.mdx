## Description

Confluence Analytics now displays the last time you visited a page or a live doc and provides an AI-generated summary of changes made since your last visit. This helps you quickly catch up on updates and stay informed about what’s new. Navigate to a Confluence page or live doc you've visited before.
	Select the Analytics dialog.
	Review the new section showing your last visit and select View changes to see the AI summary.
