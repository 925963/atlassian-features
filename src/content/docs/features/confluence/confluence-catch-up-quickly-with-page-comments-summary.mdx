## Description

Now, Atlassian Intelligence can summarize page comments for you, making it easier and faster to catch up on important feedback. Just navigate to the bottom of the page, above the first page comment, and select Summarize comments. New comments since you last checked? Select the refresh icon to generate a summary that includes all the latest feedback.
