## Description

Confluence now uses AI-generated snippets in the Following and Popular feeds. These summaries help you quickly understand what a page is about so you can decide whether it’s relevant to you. To get started with this change: Go to the Following or Popular feed in Confluence.
	Look for cards that now show an AI-generated summary of the page.
