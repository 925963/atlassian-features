## Description

This update introduces enhanced accessibility for comments with a new keyboard shortcut that lets users quickly focus on open inline comments. These improvements make it easier for everyone to navigate and interact with comments. To get started with this change: Open a page in Confluence that contains inline comments.
	Type the new keyboard shortcut to focus on open inline comments:
	
		Mac: Opt + Cmd + E
		Windows: Alt + Ctrl + E
