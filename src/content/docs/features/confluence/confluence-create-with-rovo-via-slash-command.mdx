## Description

Generate content directly from the quick insert menu in Confluence using Create with Rovo feature. This update brings discoverability to this feature and allows the user to quickly create content from the Confluence editor. To get started with this change: Open a page or blog in Confluence.
	Type / on your keyboard to open the quick insert menu.
	Select Create with Rovo from the list of options.
	Enter a prompt to generate your content.
