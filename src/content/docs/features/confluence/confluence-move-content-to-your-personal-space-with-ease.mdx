## Description

When you create content using the global create dropdown in Confluence, you'll now see a nudge suggesting you move it to your personal space. This helps you keep your work organized and ensures content doesn't accidentally end up in a shared space before it's ready. To get started with this change: Select Create from the global navigation.
	Choose a content type, such as a page, blog post, whiteboard, or database.
