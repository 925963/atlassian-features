## Description

You can now use a diagramming agent in Confluence whiteboards to quickly create mindmaps and simple flow diagrams by submitting a prompt. This feature helps you visualize ideas and processes with less manual effort. To use the new diagramming agent in Confluence whiteboards: Open a Confluence whiteboard.
	Select the option for the diagramming agent (look for a prompt or diagramming tool in the toolbar).
	Enter your prompt describing the mindmap or flow diagram you want to create.
