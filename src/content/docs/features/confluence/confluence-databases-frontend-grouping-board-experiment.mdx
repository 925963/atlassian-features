## Description

Confluence databases now offer enhanced board layout features, including grouping by user fields, subgrouping, and sorting groups. These improvements help you organize and visualize your data more effectively. To explore this change: Create a new database.
	Change the view to board layout by selecting Table layout, then choose Board layout.
	Open the Group panel in the top toolbar to group by a field, add subgroups, or sort groups.
