## Description

Focus now saves the columns you turn on or off, so your view stays the same when you come back—no need to set it up again each time. To get started with this change: Go to Focus and open your Sub-focus areas, Goals, or Work.
	Select Customize and choose the columns you want to see.
	Leave the page or refresh your browser, then return to the table. Your column preferences will still be applied.
