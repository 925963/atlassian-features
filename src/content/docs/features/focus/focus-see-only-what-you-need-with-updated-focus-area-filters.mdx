## Description

We've improved how filters work for focus areas, so you can quickly see what's relevant. Now, when you apply filters, results are shown in a simple, flat list and only the focus areas that match your filters are included. To use the improved filtering for your focus areas: Go to the list of all focus areas or sub-focus areas.
	Open the filter menu and select a filter.
