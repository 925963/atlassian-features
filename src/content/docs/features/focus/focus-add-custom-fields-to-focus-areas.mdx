## Description

Admins can now add custom fields to focus areas, giving their team more flexibility to organize and track information that matters most. To add custom fields to focus areas: Navigate to Focus admin settings.
	Select Configurations then Custom fields.
	Create custom fields as needed. You can create up to 15 fields.
