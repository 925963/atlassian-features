## Description

Now, when you hover over a work item's linked work item, you’ll see a quick summary of its key details without having to open a new tab or leave your current view. To preview linked work item details: Go to a Focus area.
	Find a work item with linked work items.
	Hover over the linked work items.
	View the details that appear.
