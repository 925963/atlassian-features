## Description

You can now sort columns in the work table, making it easier to organize and find information. This improvement helps you quickly arrange your data in the way that works best for you. Open the Focus app.
	Navigate to your focus area.
	Select the work tab
	Select the sort icon in any column header on the work table to sort the data.
