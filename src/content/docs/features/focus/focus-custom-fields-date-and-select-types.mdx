## Description

Administrators can now add new custom field types—Date, Single select, and Multi select—to Focus areas in the Focus app. This update gives teams more flexibility to capture and organize data that matches their unique business processes. To get started with this change: Go to the Focus app as an administrator.
	Select Settings, then choose Custom fields.
	Select Create custom field and choose from the new field types: Date, Single select, or Multi select.
	Configure your custom field and save you
