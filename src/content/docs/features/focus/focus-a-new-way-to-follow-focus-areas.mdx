## Description

We have introduced a new follower feature in Focus, which is consistent in experience with the current share feature. It is now easier to manage your Focus Area followers. To use the new followers feature in Focus: Open any Focus Area where you can manage followers.
	Select + (plus icon) next to the followers section.
	Add new followers or remove existing followers.
	Search for and select people to follow the content.
	Confirm your changes.
