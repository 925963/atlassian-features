## Description

Projects is now available as a platform app, meaning it's no longer a part of Atlassian Home. You can access it directly from the App Switcher in the top-left corner of any Atlassian app and add it to your app shortcuts. To find and use the Projects app: Select the app switcher (grid icon) in the top navigation bar.
	Choose Projects from the list of apps.
	Or, open Projects from the app directory in Atlassian Home.
