## Description

When visiting a Focus area summary page, you will see a new tab titled Reports which is currently in beta. When selecting the Reports tab the first time, you will see an empty state with the ability to add a report. Once you select the add button, you’ll be offered some suggestions to add. Navigate to a Focus area in your organization.
	Visit the Focus area summary page.
	Select the Reports tab.
	Generate an OKR report or link a Confluence document/page.
	Add an optional title or leave it blank
